## parse_error 원인 분석

raw json 검사 결과: 모델은 `reasoner_text` 에 풀이 본문 작성 + 답 도출 완료, 그러나 **답 추출 정규식이 실패**.

샘플 (2022-공통-12):
- 풀이 정상 완료 (방정식 인수분해 + 최댓값·최솟값 분석 + 답 도출)
- 그러나 답이 "답: N" 형식으로 명시 안됨 → 정규식 매칭 실패 → parse_error

**즉 모델 능력이 아닌 측정 환경의 한계**.
