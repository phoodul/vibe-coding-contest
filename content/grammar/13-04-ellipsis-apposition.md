## 핵심 명제

> **반복 요소는 생략 가능. 동격 that 은 명사 = 절. 관계대명사 that 은 명사 ↔ 불완전한 절.**

생략 룰과 동격·관계대명사 구별을 한 단원에.

---

## 설명

### 생략 — 반복 요소 빼기

영어는 같은 구조에서 반복 요소를 자주 생략.

**1) 등위접속사 후 반복 요소**

```
✓ He works in Seoul, and (he works) in Busan too.
✓ I want to go and (I want to) study English.
```

**2) be 동사 + 분사**

```
✓ The man (who is) standing there is my dad.
✓ The book (which was) written by him is great.
```

(8-2 단원 관계대명사 + be 동사 함께 생략)

**3) 부사절의 주어 + be 동사**

```
✓ When (you are) in Rome, do as the Romans do.
✓ Though (he was) tired, he kept working.
✓ If (it is) necessary, call me.
```

주절 주어와 같은 부사절의 주어·be 흔히 생략.

**4) to부정사의 동사 생략 (대부정사 to)**

```
A: Do you want to go?
B: Yes, I'd love to. (= I'd love to go. — go 생략)
```

`to` 만 남기고 동사 생략.

**5) 비교 절 안의 동사 반복 생략**

```
✓ He is taller than I (am).
✓ She runs faster than he (does).
```

### 동격 — 명사 = 명사·절

명사를 다른 명사·절·구로 다시 설명.

**1) 명사 + 명사 (콤마)**

```
✓ My friend, Tom, is a doctor. (My friend = Tom)
✓ Seoul, the capital of Korea, is busy.
```

**2) 명사 + that 절**

```
✓ The fact that he lied surprised me. (the fact = he lied)
✓ The news that the team won is true.
```

`fact / news / idea / belief / hope / rumor` 같은 명사 뒤에 that 절.

### 동격 that vs 관계대명사 that — 결정적 차이

| 종류 | that 절 안의 구조 | 의미 |
|---|---|---|
| 동격 that | **완전한 절** (명사 = 절) | ~라는 사실/뉴스 |
| 관계대명사 that | **불완전한 절** (주어/목적어 빠짐) | 명사를 수식 |

```
✓ The fact that he lied surprised me. (he lied 완전 → 동격)
✓ The book that I read is great. (I read ___ 목적어 빈 → 관계대명사)
```

### 실전 구별 — 절 안에 빈 자리가 있나?

```
The fact that he lied → "he lied" 완전 → 동격
The news that I heard → "I heard ___" 목적어 빈 → 관계대명사
```

같은 that 인데 둘 다 영어로 동일 형태. 의미·구조로만 구별.

### 동격 that 받는 명사들 (외우기)

`fact`, `news`, `idea`, `belief`, `hope`, `rumor`, `truth`, `opinion`, `thought`, `discovery`, `confession`, `proof`.

```
✓ I have no idea that he was here.
✓ The hope that we can succeed inspires us.
```

### that 외 동격 — of + 명사 / to부정사 / 콤마

```
✓ The idea of leaving early appealed to me. (of + V-ing)
✓ The decision to leave was difficult. (to V)
✓ My brother, a doctor, lives in Seoul. (콤마)
```

## 대표 문장 (외우기)

> **The fact that he lied surprised me — but the book that I read is still great.**
>
> 그가 거짓말한 사실이 나를 놀라게 했다 — 그러나 내가 읽은 책은 여전히 훌륭하다.

`The fact that he lied` (동격 — 완전한 절) + `the book that I read` (관계대명사 — 목적어 빈). 두 that 의 차이.

## 실전 문제 5개

### 문제 1.
다음 빈칸에 알맞은 것은? (생략 가능)

`When _____ in Rome, do as the Romans do.`

① you  ② you are  ③ to be  ④ being

### 문제 2.
다음 중 동격 that 인 것은?

① The book that I bought is interesting.
② The fact that he is honest is well known.
③ The man that I met is kind.
④ The car that broke down is mine.

### 문제 3.
어법상 옳은 것은?

① The man standing there is my brother.
② The man who standing there is my brother.
③ The man is standing there is my brother.
④ The man, that is standing there, is my brother.

### 문제 4.
다음 두 문장이 같은 의미가 되도록 빈칸에 알맞은 것은?

`I have a hope. We can succeed.` (두 문장 → 한 문장)

① I have a hope which we can succeed.
② I have a hope that we can succeed.
③ I have a hope, that we can succeed.
④ I have a hope to we can succeed.

### 문제 5.
다음 빈칸에 알맞은 것은?

`A: Do you want to go?`
`B: Yes, I'd love _____ .`

① to  ② go  ③ to go  ④ ① 과 ③ 모두 가능

## 정답 + 해설

### 1. ② you are

부사절 주어 + be 생략 가능 → `When in Rome` 도 OK. 답은 보존 형태 ②.

### 2. ② The fact that he is honest is well known.

`he is honest` 완전한 절 + the fact = 그 사실 → 동격. 나머지는 모두 관계대명사 (불완전한 절).

### 3. ① The man standing there is my brother.

`who is` 생략 → standing 분사. ② is 누락. ③ 두 동사 X. ④ 콤마 + that 계속적 X (8-5).

### 4. ② I have a hope that we can succeed.

hope 는 동격 that 받는 명사.

### 5. ④ ① 과 ③ 모두 가능

대부정사 to (동사 생략) 또는 to + 동사 모두 OK.
