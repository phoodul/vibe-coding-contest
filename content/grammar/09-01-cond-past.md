## 핵심 명제

> **현재 사실의 반대 = If 주어 + 과거동사, 주어 + would/could/might + 원형.**

if 절 안의 be 동사는 모두 were (was 도 OK 지만 표준은 were).

---

## 설명

### 4가지 if 패턴 — 한눈에

| 종류 | if 절 시제 | 주절 시제 | 의미 |
|---|---|---|---|
| 단순 조건 (0/1형) | 현재 | will + 원형 | 미래 가능성 (실현 가능) |
| **가정법 과거** | 과거 | would + 원형 | **현재 사실 반대** |
| **가정법 과거완료** | 과거완료 | would have + p.p. | 과거 사실 반대 (9-2) |
| 가정법 미래 | should/were to + V | will/would + V | 미래 불확실 (격식) |

### 가정법 과거 — 현재 사실의 반대

지금 일어나지 않는 일을 가정.

```
✓ If I had a million dollars, I would buy a house.
(돈이 없다 → 만약 있다면, 집을 살 텐데)

✓ If I were you, I wouldn't do that.
(나는 너가 아니다 → 만약 너라면, 그러지 않을 텐데)

✓ If she knew the truth, she would be sad.
(그녀는 진실을 모른다 → 만약 안다면, 슬플 텐데)
```

### if 절 안의 be 동사 = were (모든 인칭)

가정법 과거에서 be 동사는 인칭 무관 **were**.

```
✓ If I were you, ... (격식·표준)
✓ If I was you, ... (구어 OK)
✓ If he were rich, ...
```

표준은 were. 시험에선 were 가 답.

### 단순 조건 vs 가정법 과거 — 의미 차이

```
단순 조건 (실현 가능):
If I have time, I will help you.
(시간 있으면, 도와줄게 — 시간 있을 가능성 50%)

가정법 과거 (반대):
If I had time, I would help you.
(시간 없다 → 만약 있다면 도와줄 텐데)
```

같은 한국어 "시간 있으면" 인데 영어로는 정반대 의미를 표현 가능.

### 주절의 조동사 — would / could / might

| 조동사 | 의미 |
|---|---|
| would | ~할 텐데 (의지·예측) |
| could | ~할 수 있을 텐데 (능력) |
| might | ~할지 모를 텐데 (약한 추측) |

```
✓ If I knew the answer, I would tell you. (알려줄 의지)
✓ If I knew the answer, I could tell you. (알려줄 수 있음)
✓ If I knew the answer, I might tell you. (알려줄지 모름)
```

### 가정법 = 직설법 변환 — 시험 단골

```
가정법 과거: If I were rich, I would travel the world.
직설법 변환: As I am not rich, I cannot travel the world.
(또는: I am not rich, so I cannot travel.)
```

가정법 과거 = 현재의 부정 사실. 변환 시 시제 + 부정.

### 다양한 if 표현 (확장)

| 표현 | 의미 |
|---|---|
| If only | ~라면 좋을 텐데 |
| What if ~? | ~면 어떨까? |
| Suppose / Supposing | ~라고 가정해봐 |
| Imagine | ~을 상상해봐 |

```
✓ If only I were taller! (더 키 컸으면 좋을 텐데)
✓ What if you were the president?
```

### 도치형 — Were I you (9-6 에서)

if 생략 + 도치 가능.

```
✓ Were I you, I wouldn't do that.
= If I were you, I wouldn't do that.
```

(자세히는 9-6 `cond-inversion`)

## 대표 문장 (외우기)

> **If I were rich, I would travel the world. — But as I am not rich, I cannot.**
>
> 만약 내가 부자라면 세계 일주를 할 텐데. 하지만 부자가 아니라서 할 수 없다.

`If I were rich, I would travel` (가정법 과거) + 직설법 변환. 핵심 패턴.

## 실전 문제 5개

### 문제 1.
어법상 옳은 것은?

① If I am you, I would not do that.
② If I were you, I would not do that.
③ If I was you, I will not do that.
④ If I were you, I will not do that.

### 문제 2.
다음 가정법 문장의 직설법 변환으로 옳은 것은?

`If I had wings, I would fly to you.`

① I have wings, so I will fly to you.
② I don't have wings, so I can't fly to you.
③ I had wings, so I flew to you.
④ I will have wings, so I will fly to you.

### 문제 3.
다음 빈칸에 알맞은 것은?

`If she _____ the truth, she would be sad.`

① know  ② knew  ③ has known  ④ will know

### 문제 4.
다음 중 어법상 **틀린** 문장은?

① If I were taller, I would play basketball.
② If she had time, she would help you.
③ If I will know it, I will tell you.
④ If he were here, he would explain it.

### 문제 5.
다음 우리말을 영어로 옮길 때 가장 적절한 것은?

> 내가 새라면 너에게 날아갈 텐데.

① If I am a bird, I will fly to you.
② If I were a bird, I would fly to you.
③ If I have been a bird, I would fly to you.
④ If I will be a bird, I would fly to you.

## 정답 + 해설

### 1. ② If I were you, I would not do that.

가정법 과거 = If + were + would + 원형. ① ③ ④ 시제 X.

### 2. ② I don't have wings, so I can't fly to you.

가정법 과거 = 현재 부정. "If I had wings (지금 없음)" → "I don't have wings".

### 3. ② knew

가정법 과거 = if + 과거. 주절이 would be 인 점도 단서.

### 4. ③ If I will know it, I will tell you.

조건 부사절은 미래도 현재 → `If I know it, I will tell you.` (단순 조건). 가정법 과거이면 `If I knew, I would tell`.

### 5. ② If I were a bird, I would fly to you.

가정법 과거 + were + would.
