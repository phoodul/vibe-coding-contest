## 핵심 명제

> **should = 충고·당연 (그래야 한다, 70%). had better = 더 강한 충고 (안 하면 후회).**

3가지 강도를 구별하라.

---

## 설명

### should — 충고·예측

**1) 충고 (~해야 한다)** — 약한 의무

`You should study harder.` (공부 좀 더 해야 한다 — 충고)
`We should leave now.`

must 보다 약함, 강요가 아닌 권고.

**2) 합리적 예측 (~일 것이다, 70%)**

`He should be home by now.` (지금쯤 집에 있을 것 같다)
`The package should arrive tomorrow.`

### ought to — should 와 거의 동의어

```
✓ You ought to apologize. (= should apologize)
✓ He ought to know better.
```

차이:
- ought to 는 약간 격식
- ought to 는 to 가 붙음 (다른 조동사와 다름)
- 부정 — `ought not to` (드물게 사용, should not 가 더 흔함)

```
✓ He ought not to lie.
✓ He shouldn't lie. (더 흔함)
```

### had better — 강한 충고 / 경고

`had better + 동사원형` = 안 하면 안 좋은 결과 발생할 거라는 경고.

`You had better see a doctor.` (의사 보러 가는 게 좋다 — 안 가면 위험)
`We had better leave now or we'll be late.`

축약: `'d better`. 부정: `had better not + 원형`.

```
✓ You'd better not be late.
✗ You'd not better be late.
```

`had better` 는 형태가 had + better 지만 의미는 **현재·미래**. 과거 의미 X.

### should + have + p.p. — 했어야 했는데 (후회)

과거에 했어야 할 일을 안 한 것에 대한 후회.

`I should have studied harder.` (공부 더 했어야 했는데 — 안 했음)
`You shouldn't have lied.` (거짓말 하지 말았어야 했다 — 했음)

(자세히는 4-6 `modal-perfect`)

### 충고 강도 정리

가장 약함 → 가장 강함:

| 표현 | 강도 | 어감 |
|---|---|---|
| might | 매우 약 | 그래도 됨 |
| could | 약 | 가능성 제안 |
| should / ought to | 중 | 권고·충고 |
| had better | 강 | 안 하면 위험 |
| must / have to | 매우 강 | 의무 |

### 가정법 should — 만약 ~라면 (격식)

`Should you have any questions, please contact me.`
(혹시 질문이 있으시면 — `If you should have any questions` 의 도치)

격식 표현. (단원 9 가정법 + 단원 13 도치에서)

## 대표 문장 (외우기)

> **You should drink water, but you'd better not drink alcohol before driving.**
>
> 물은 마시는 게 좋다. 그러나 운전 전에 술은 절대 마시지 않는 게 좋다.

`should` (일반 충고) + `had better not` (강한 경고). 충고 강도 두 단계.

## 실전 문제 5개

### 문제 1.
다음 빈칸에 알맞은 것은?

`You _____ see a doctor. You look terrible.`

① must  ② should  ③ might  ④ can

### 문제 2.
어법상 옳은 것은?

① You'd not better be late.
② You'd better not be late.
③ You hadn't better be late.
④ You don't better be late.

### 문제 3.
의미가 가장 비슷한 것은?

`We ought to apologize.`

① We should apologize.
② We must apologize.
③ We can apologize.
④ We may apologize.

### 문제 4.
빈칸에 들어갈 적절한 것은?

`I _____ studied harder for the exam. I failed.`

① should  ② should have  ③ should had  ④ should had been

### 문제 5.
다음 우리말을 영어로 옮길 때 가장 적절한 것은?

> 안 하면 후회할 거야. 지금 떠나는 게 좋겠어.

① You should leave now or you will regret it.
② You had better leave now or you will regret it.
③ You may leave now or you will regret it.
④ You can leave now or you will regret it.

## 정답 + 해설

### 1. ② should

충고 → should. ① must 너무 강함. ③ might 약 추측. ④ can 능력·허가.

### 2. ② You'd better not be late.

`had better not + 원형`. ① 어순 X. ③ hadn't better X. ④ don't + better X.

### 3. ① We should apologize.

ought to ≒ should. ② must 더 강함. ③ ④ 의미 다름.

### 4. ② should have

과거 했어야 했는데 안 한 후회 → should have + p.p. ① 현재 충고. ③ should had X. ④ 형태 X.

### 5. ② You had better leave now or you will regret it.

"안 하면 후회" 강한 경고 → had better. ① should 일반 충고. ③ may / ④ can 충고 의미 X.
