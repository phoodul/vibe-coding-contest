## 핵심 명제

> **used to = 과거 상태·반복 동작 (지금은 X). would = 과거 반복 동작 (상태 X).**

상태에 would 못 쓴다 — 가장 큰 차이.

---

## 설명

### used to — 과거의 상태·습관 (둘 다)

지금은 더 이상 그렇지 않은 과거의 일.

**1) 상태**:
`I used to be shy.` (예전엔 내성적이었다 — 지금은 X)
`There used to be a tree here.` (예전에 나무 한 그루가 있었다)

**2) 반복 동작**:
`I used to play soccer every weekend.` (예전엔 매주 축구했다)

### would — 과거의 반복 동작만 (상태 X)

```
✓ When I was a child, I would visit my grandmother every Sunday. (반복 동작)
✗ When I was a child, I would be shy. (상태 X)
```

상태에 would 쓰면 어색. used to 만 가능.

### 형태 — 부정·의문

| 표현 | 부정 | 의문 |
|---|---|---|
| used to | didn't use to / used not to | Did you use to? |
| would | wouldn't | Would you? |

```
✓ I didn't use to like coffee.
✓ Did you use to live here?
✓ I wouldn't visit them often.
```

`use to` (과거형 -d 빠짐) 도 OK 지만 `used to` 가 표준.

### would 의 다른 의미

would 는 과거 습관 외에도 다양:

**1) 정중한 부탁** — `Would you...?`
- `Would you pass the salt?` (소금 좀 건네주세요)

**2) 의지·고집의 과거** — `would not + 원형`
- `She wouldn't listen to me.` (말을 안 들었다)

**3) 가정법 — 단원 9**
- `If I were rich, I would buy a car.`

**4) 시제 일치 — will 의 과거**
- `He said he would come.` (= He says he will come.)

### be used to V-ing — 익숙해진

`used to + 원형` 과 형태가 비슷한 함정.

| 표현 | 의미 |
|---|---|
| used to + 원형 | 과거 습관 (지금은 X) |
| be used to + V-ing/명사 | ~에 익숙해진 |
| get used to + V-ing/명사 | ~에 익숙해지다 |

```
✓ I used to live in Seoul. (예전 서울 살았음 — 지금은 X)
✓ I am used to living in Seoul. (서울 사는 데 익숙)
✓ I am getting used to spicy food. (매운 음식에 익숙해지는 중)
```

세 형태 다 시험 단골. **be used to 뒤는 동명사 또는 명사** (to + V 원형 X).

```
✗ I am used to live in Seoul.
✓ I am used to living in Seoul.
```

## 대표 문장 (외우기)

> **I used to be shy, but now I'm used to speaking in public.**
>
> 예전엔 내성적이었는데, 지금은 사람들 앞에서 말하는 데 익숙해.

`used to be` (과거 상태) + `am used to speaking` (현재 익숙). 두 함정이 한 문장에.

## 정답 + 해설

### 1. ① used to live in

상태도 가능 + 동사 원형. ② would 반복 동작만 (살다는 상태) X. ③ am used to + V-ing 형태 X. ④ used to + lived X.

### 2. ② I am used to living in cold weather.

be used to + V-ing/명사. ① to + 원형 X. ③ to + lived X. ④ to 누락.

### 3. ② When I was young, I would be very shy.

would 는 상태 X. → `I used to be very shy.` 가 정상. 나머지 정상.

### 4. ③ I am used to drinking coffee.

`be used to + V-ing` = 익숙. 나머지 3개는 모두 과거 습관 (지금은 안 마심) 의미.

### 5. ② He used to live here, but now he lives elsewhere.

과거 거주 (지금은 X) → used to + 원형. ① uses to 형태 X. ③ is used to + 원형 X. ④ would + lives 형태 X 게다가 거주는 상태.
