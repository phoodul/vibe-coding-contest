## 핵심 명제

> **명사가 동작을 하면 -ing (능동), 동작을 받으면 -ed (수동).**

분사 선택 = "주체인가, 대상인가" 한 가지 질문.

---

## 설명

분사는 동사를 형용사처럼 쓴 것. 두 형태:
- 현재분사 (V-ing): 능동·진행 (~하는)
- 과거분사 (V-ed/p.p.): 수동·완료 (~된)

### 명사 수식 — 위치

분사 한 단어면 명사 **앞**, 분사 + 부속어면 명사 **뒤**.

```
✓ a sleeping baby (자고 있는 아기 — 한 단어 앞)
✓ a baby sleeping in the cradle (요람에서 자는 아기 — 부속 뒤)

✓ a broken vase (깨진 화분)
✓ a vase broken by my cat (내 고양이가 깬 화분)
```

### 능동 vs 수동 — 명사가 행위자인가 대상인가

```
✓ a barking dog (짖는 개 — 개가 짖는 주체)
✓ a beaten dog (맞은 개 — 개가 맞는 대상)

✓ falling leaves (떨어지는 나뭇잎 — 잎이 떨어지는 주체)
✓ fallen leaves (떨어진 나뭇잎 — 이미 떨어진 결과)
```

### 자동사의 과거분사 — 완료 의미

자동사 (목적어 X) 의 -ed 는 수동 X, **완료** 의미.

```
✓ fallen leaves (떨어진 — 완료)
✓ retired teacher (은퇴한 — 완료)
✓ deceased person (사망한 — 완료)
```

### 감정 형용사 — 가장 헷갈리는 부분

`interesting / interested`, `boring / bored` 같은 감정 형용사. (1-5 단원에서 다룸)

| -ing | -ed |
|---|---|
| 그것이 ~하는 (감정을 줌) | ~한 감정을 받는 |
| **사물·상황** 묘사 | **사람** 의 감정 |
| an interesting book | an interested student |
| a boring movie | a bored audience |
| an exciting game | excited fans |

```
✗ I am boring. (= 나는 지루한 사람)
✓ I am bored. (= 내가 지루함을 느낀다)
```

자주 시험에 나오는 짝:

| -ing 형 | -ed 형 |
|---|---|
| interesting | interested |
| boring | bored |
| exciting | excited |
| surprising | surprised |
| confusing | confused |
| disappointing | disappointed |
| frightening | frightened |
| shocking | shocked |
| tiring | tired |
| amazing | amazed |

### 분사가 명사 뒤에서 길게 수식 — 절 단축

관계대명사 + be 동사를 빼고 분사만 남긴 형태.

```
The man who is standing over there is my father.
→ The man standing over there is my father. (현재분사)

The book which was written by him is famous.
→ The book written by him is famous. (과거분사)
```

(자세히는 단원 8 관계사 + 7-2 분사구문)

### 명사 수식 vs 보어 — 분사 모양은 같음

```
✓ a sleeping baby (수식)
✓ The baby is sleeping. (be V-ing — 진행 보어)

✓ a broken window (수식)
✓ The window is broken. (수동 보어)
```

문장 안에서 위치만 다름.

## 대표 문장 (외우기)

> **The interesting book made me interested in history.**
>
> 그 흥미로운 책이 나를 역사에 관심 있게 만들었다.

`interesting book` (사물 — 감정 줌) + `interested student` (사람 — 감정 받음). 가장 자주 틀리는 짝.

## 실전 문제 5개

### 문제 1.
다음 빈칸에 알맞은 것은?

`Look at the _____ baby.`

① sleep  ② sleeping  ③ slept  ④ sleeps

### 문제 2.
어법상 옳은 것은?

① The movie was very bored.
② The movie was very boring.
③ The movie was very bore.
④ The movie was bored to me.

### 문제 3.
다음 중 어법상 **틀린** 문장은?

① The leaves fallen on the ground are red.
② The boy standing there is my brother.
③ This is a book wrote by Hemingway.
④ I am interested in classical music.

### 문제 4.
빈칸에 들어갈 적절한 것은?

`The students were _____ by the surprising news.`

① shock  ② shocking  ③ shocked  ④ shocks

### 문제 5.
다음 우리말을 영어로 옮길 때 가장 적절한 것은?

> 그는 그 결과에 실망했다.

① He was disappointing with the result.
② He was disappointed with the result.
③ He was disappointed by disappointing.
④ He was disappoint with the result.

---

## 정답 + 해설

### 1. ② sleeping

자는 동작을 하는 주체 = 능동 → -ing.

### 2. ② The movie was very boring.

영화가 지루함을 줌 (사물) → -ing. ① bored 는 사람이 받는 감정.

### 3. ③ This is a book wrote by Hemingway.

`written` (과거분사) 이 정상. wrote 는 단순과거. → `a book written by Hemingway`.

### 4. ③ shocked

학생들이 (사람) 충격 받음 → -ed.

### 5. ② He was disappointed with the result.

사람이 실망함 → -ed.
