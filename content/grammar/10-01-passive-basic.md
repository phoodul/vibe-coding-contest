## 핵심 명제

> **목적어 → 주어 자리. 동사 → be + p.p. 행위자 → by + 목적격.**

3 단계 변환만 외우면 된다.

---

## 설명

수동태 = "행위를 받는 쪽" 을 강조하는 문장 형식.

```
능동: She wrote the letter.
       (그녀가 / 썼다 / 편지를)
수동: The letter was written by her.
       (편지가 / 쓰여졌다 / 그녀에 의해)
```

### 변환 3 단계

1. 능동의 **목적어 → 수동의 주어**
2. 능동의 **동사 → be + 과거분사** (시제는 능동과 같게)
3. 능동의 **주어 → by + 목적격**

```
능동: Tom broke the window.
1. window → 주어
2. broke (과거) → was broken (과거 수동)
3. Tom → by Tom
수동: The window was broken by Tom.
```

### 시제별 수동태 형태

| 시제 | 형태 |
|---|---|
| 단순현재 | am/is/are + p.p. |
| 단순과거 | was/were + p.p. |
| 미래 | will be + p.p. |
| 현재진행 | am/is/are being + p.p. |
| 과거진행 | was/were being + p.p. |
| 현재완료 | have/has been + p.p. |
| 과거완료 | had been + p.p. |
| 미래완료 | will have been + p.p. |
| 조동사 | 조동사 + be + p.p. |

```
✓ The book is read by many. (현재)
✓ The book was read by many. (과거)
✓ The book will be read by many. (미래)
✓ The book has been read by many. (현재완료)
✓ The book is being read now. (현재진행)
✓ The book can be read by anyone. (조동사)
```

### by + 행위자 — 생략되는 경우

다음 4 케이스에선 by 행위자 흔히 생략:

| 경우 | 예 |
|---|---|
| 행위자 불명·일반인 | English is spoken here. |
| 행위자 명확 | The thief was arrested. (당연히 경찰) |
| 행위자 불특정 | Many books are sold every day. |
| 강조할 필요 X | He was injured in the accident. |

```
✓ The window was broken. (누가 깼는지 불명·중요 X)
✗ The window was broken by someone. (불필요)
```

### by 외 다른 전치사

일부 동사·형용사는 by 가 아닌 다른 전치사.

| 패턴 | 전치사 |
|---|---|
| be interested in | in |
| be surprised at | at |
| be pleased with | with |
| be satisfied with | with |
| be filled with | with |
| be made of | of (재료 보임) |
| be made from | from (재료 안 보임) |
| be covered with | with |
| be known to | to (사람) |
| be known for | for (이유) |
| be known as | as (~로서) |

```
✓ I am interested in music. (interested in 고정)
✓ The bottle is filled with water. (filled with)
✓ He is known to everyone. (known to)
```

(자세히는 10-4 `passive-prepositions`)

### 수동태가 부적절한 경우

자동사는 수동태 X (목적어 X 라 변환 불가).

```
✗ The accident was happened.
✓ The accident happened. (자동사 — 수동 X)

✗ He was died.
✓ He died.
```

`happen, occur, die, arrive, exist, disappear` 등은 모두 자동사.

(10-5 `passive-intransitive` 에서 자세히)

## 대표 문장 (외우기)

> **The letter was written by her, and it will be sent tomorrow.**
>
> 그 편지는 그녀에 의해 쓰여졌고, 내일 보내질 것이다.

`was written by her` (과거 수동) + `will be sent` (미래 수동). 두 시제.

---

## 실전 문제 5개

### 문제 1.
다음 능동 문장을 수동태로 가장 자연스럽게 바꾼 것은?

`Tom broke the window.`

① The window broke by Tom.
② The window was broken by Tom.
③ The window has broken by Tom.
④ The window is breaking by Tom.

### 문제 2.
어법상 옳은 것은?

① The cake is making by my mom.
② The cake is being made by my mom.
③ The cake is be made by my mom.
④ The cake is being making by my mom.

### 문제 3.
다음 중 어법상 **틀린** 문장은?

① The accident was happened yesterday.
② This book was written in 1990.
③ Many people are killed in war.
④ The package will be delivered tomorrow.

### 문제 4.
빈칸에 들어갈 적절한 것은?

`The bottle is filled _____ water.`

① by  ② of  ③ with  ④ from

### 문제 5.
다음 우리말을 영어로 옮길 때 가장 적절한 것은?

> 이 편지는 어제 그녀가 보냈다.

① This letter was sent by her yesterday.
② This letter is sent by her yesterday.
③ This letter has been sent by her yesterday.
④ This letter sent by her yesterday.

---

## 정답 + 해설

### 1. ② The window was broken by Tom.

과거 수동 = was/were + p.p.

### 2. ② The cake is being made by my mom.

현재진행 수동 = is being + p.p.

### 3. ① The accident was happened yesterday.

happen 은 자동사 → 수동 X. → `The accident happened yesterday.`

### 4. ③ with

filled with (고정).

### 5. ① This letter was sent by her yesterday.

과거 수동 + yesterday.
