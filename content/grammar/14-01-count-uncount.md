## 핵심 명제

> **가산 = many/few/(a) few, 불가산 = much/little/(a) little. a 유무가 긍정·부정 갈라.**

수량 형용사는 명사 종류와 관사로 결정.

---

## 설명

(1-1 `parts-noun` 의 가산·불가산 룰 복습 + 수량 형용사 정리.)

### 수량 형용사 한눈에

| 의미 | 가산 | 불가산 | 둘 다 |
|---|---|---|---|
| 많은 (긍정) | many | much | a lot of / lots of / plenty of |
| 약간의 (긍정) | a few | a little | some |
| 거의 없는 (부정) | few | little | hardly any |

### many / much — 많은

```
✓ I have many friends. (가산)
✓ I have much money. (불가산)
```

구어에선 `a lot of` / `lots of` 가 더 흔함.

```
✓ I have a lot of friends.
✓ I have a lot of money.
```

### a few / few — 미묘한 차이 (가산)

| 형태 | 의미 |
|---|---|
| a few + 가산 복수 | 약간 있다 (긍정) |
| few + 가산 복수 | 거의 없다 (부정) |

```
✓ I have a few friends. (몇 명 있다 — 좋은 의미)
✓ I have few friends. (거의 없다 — 외로움)
```

`a` 한 글자가 정반대 의미.

### a little / little — 미묘한 차이 (불가산)

| 형태 | 의미 |
|---|---|
| a little + 불가산 | 약간 있다 (긍정) |
| little + 불가산 | 거의 없다 (부정) |

```
✓ I have a little money. (좀 있다)
✓ I have little money. (거의 없다)
```

### a number of / the number of — 함정

| 표현 | 의미 | 동사 |
|---|---|---|
| a number of + 복수 | 많은 | 복수 |
| the number of + 복수 | ~의 수 | 단수 |

```
✓ A number of students are absent. (많은 학생들이 결석 — 복수)
✓ The number of students is decreasing. (학생 수가 감소 — 단수)
```

a vs the 만 다른데 동사가 달라지는 함정.

### 자주 시험에 나오는 함정 명사

| 명사 | 가산? |
|---|---|
| information | 불가산 (✗ informations) |
| advice | 불가산 (✗ advices) |
| furniture | 불가산 |
| equipment | 불가산 |
| homework | 불가산 |
| money | 불가산 |
| news | 불가산 |
| baggage / luggage | 불가산 |
| weather | 불가산 |
| evidence | 불가산 |

```
✗ I have many informations.
✓ I have much information.

✗ I gave him many advices.
✓ I gave him much advice.
```

(1-1 단원에서 자세히)

### 단위 표현 — 불가산을 셀 때

| 단위 | 명사 |
|---|---|
| a piece of | advice, information, news, furniture, paper |
| a glass of | water, milk |
| a slice of | bread, cake |
| a sheet of | paper |
| a bar of | chocolate, soap |
| a loaf of | bread |
| a bottle of | water, wine |

```
✓ a piece of advice (충고 한 마디)
✓ two glasses of water (물 두 잔)
```

### 둘 다 — some / any

`some` 과 `any` 는 가산·불가산 모두 가능. some 은 긍정·권유, any 는 부정·의문.

```
✓ I have some books / some money.
✓ I don't have any books / any money.
✓ Do you have any questions?

✓ Would you like some tea? (권유 — 긍정 어감)
```

## 대표 문장 (외우기)

> **I have a few friends and a little money — but few enemies and little time.**
>
> 친구 몇 명과 돈 좀 있다 — 그러나 적은 적과 부족한 시간.

`a few friends` (긍정) + `few enemies` (부정) + `a little money / little time` (불가산 긍/부정). 4가지 차이.

## 실전 문제 5개

### 문제 1.
다음 빈칸에 알맞은 것은?

`I have _____ books in my bag.`

① much  ② many  ③ a little  ④ little

### 문제 2.
다음 두 문장의 의미 차이로 알맞은 것은?

`A: I have a few friends.`
`B: I have few friends.`

① 둘 다 친구 많음
② A 는 친구 좀 있음, B 는 거의 없음
③ A 는 거의 없음, B 는 좀 있음
④ 둘 다 친구 없음

### 문제 3.
어법상 옳은 것은?

① The number of students are increasing.
② The number of students is increasing.
③ A number of students is absent.
④ A number of students were absents.

### 문제 4.
다음 중 어법상 **틀린** 문장은?

① I have many informations.
② I have a piece of advice for you.
③ I drank some water.
④ I have a lot of money.

### 문제 5.
다음 우리말을 영어로 옮길 때 가장 적절한 것은?

> 그는 충고를 거의 안 한다.

① He gives many advice.
② He gives few advice.
③ He gives little advice.
④ He gives any advices.

## 정답 + 해설

### 1. ② many

books 가산 복수 → many.

### 2. ② A 는 친구 좀 있음, B 는 거의 없음

a few (긍정) vs few (부정).

### 3. ② The number of students is increasing.

`The number of` = ~의 수 → 단수 동사 is.

### 4. ① I have many informations.

information 불가산 → -s X, many X. → `much information` 또는 `a lot of information`.

### 5. ③ He gives little advice.

advice 불가산 → little (부정 의미). few 는 가산. -s 도 X.
