## 핵심 명제

> **If 절은 가정법 과거완료, 주절은 가정법 과거 = 혼합 가정법.**

과거의 일이 현재까지 영향을 주는 가정.

---

## 설명

### 혼합 가정법 — 과거 원인 + 현재 결과

if 절이 **과거 사실**의 반대를 가정 + 주절이 **그로 인한 현재**의 반대를 표현.

```
✓ If I had studied harder, I would be a doctor now.
(과거에 공부 안 했다 → 지금 의사 아니다.
 만약 그때 공부했다면, 지금 의사일 텐데.)

✓ If she had taken the train, she would be here now.
(기차 안 탔다 → 지금 여기 없다.
 만약 탔다면, 지금 여기 있을 텐데.)
```

### 형태 — 한눈에

| 절 | 시제 |
|---|---|
| if 절 | had + p.p. (가정법 과거완료) |
| 주절 | would + 원형 (가정법 과거) |

핵심 구별 단서: 주절에 **now / today / these days** 같은 현재 부사가 자주 등장.

```
✓ If I had been born in the US, I would speak English fluently now.
✓ If you had eaten breakfast, you wouldn't be hungry now.
```

### 일반 가정법 과거완료 vs 혼합 — 시제 비교

```
가정법 과거완료 (과거 결과):
If I had studied harder, I would have passed the exam (last year).
(과거 시점에 합격했을 텐데)

혼합 가정법 (현재 결과):
If I had studied harder, I would be a doctor now.
(지금 의사일 텐데)
```

주절의 시점이 과거인지 현재인지가 결정.

### 직설법 변환

```
혼합: If I had studied harder, I would be a doctor now.
직설: I didn't study harder, so I am not a doctor now.
```

과거 부정 + 현재 부정.

### 자주 출제되는 표현

```
✓ If I hadn't missed the bus, I would be at school now.
✓ If she had taken the medicine, she would feel better today.
✓ If you had told me earlier, I would know what to do.
```

### now / today 가 단서

주절에 시간 부사가 있으면 거의 혼합 가정법.

```
주절에 now → 혼합 (현재 결과)
주절에 yesterday → 가정법 과거완료 (과거 결과)
```

---

## 대표 문장 (외우기)

> **If I had studied harder, I would be a doctor now.**
>
> 내가 더 열심히 공부했다면, 지금 의사일 텐데.

`had studied` (과거 가정 — 가정법 과거완료) + `would be a doctor now` (현재 결과 — 가정법 과거). 혼합의 표준.

## 실전 문제 5개

### 문제 1.
다음 빈칸에 알맞은 것은?

`If I _____ the lottery last year, I _____ rich now.`

① won, would be  ② had won, would be  ③ won, will be  ④ have won, would be

### 문제 2.
다음 문장의 직설법 변환으로 옳은 것은?

`If she had married him, she would be happy now.`

① She married him, so she is happy now.
② She didn't marry him, so she is not happy now.
③ She doesn't marry him, so she is not happy now.
④ She had married him, so she is happy.

### 문제 3.
어법상 옳은 것은?

① If I had eaten breakfast, I am not hungry now.
② If I had eaten breakfast, I would not be hungry now.
③ If I ate breakfast, I would not be hungry now.
④ If I had eaten breakfast, I will not be hungry now.

### 문제 4.
다음 중 혼합 가정법인 것은?

① If it rains tomorrow, I will stay home.
② If I were rich, I would travel.
③ If she had taken the bus, she would be here now.
④ If he had studied, he would have passed.

### 문제 5.
다음 우리말을 영어로 옮길 때 가장 적절한 것은?

> 만약 내가 그때 약을 먹었다면, 지금 아프지 않을 텐데.

① If I take the medicine then, I am not sick now.
② If I took the medicine then, I would not be sick now.
③ If I had taken the medicine then, I would not be sick now.
④ If I had taken the medicine then, I would not have been sick now.

## 정답 + 해설

### 1. ② had won, would be

last year (과거 가정) + now (현재 결과) → 혼합. if had p.p. + would 원형.

### 2. ② She didn't marry him, so she is not happy now.

혼합 = 과거 부정 + 현재 부정.

### 3. ② If I had eaten breakfast, I would not be hungry now.

혼합 = had p.p. + would 원형. ① 주절 시제 X. ③ 가정법 과거 (둘 다 현재). ④ will X.

### 4. ③ If she had taken the bus, she would be here now.

if 과거완료 + 주절 가정법 과거 + now → 혼합. ① 단순 조건. ② 가정법 과거. ④ 가정법 과거완료.

### 5. ③ If I had taken the medicine then, I would not be sick now.

then (과거) + now (현재) → 혼합.
