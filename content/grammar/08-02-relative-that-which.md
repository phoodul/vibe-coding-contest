## 핵심 명제

> **사물 선행사 + 한정적 = which 또는 that. 계속적 (콤마 ,) = which 만, that X.**

콤마가 결정한다.

---

## 설명

### 사물 선행사용 관계대명사

| 격 | 형태 |
|---|---|
| 주격 | which / that |
| 목적격 | which / that / 생략 |
| 소유격 | whose / of which |

```
✓ The book which is on the table is mine. (주격)
✓ The book that I read yesterday is interesting. (목적격)
✓ The book (which/that) I bought is expensive. (생략)
```

### 한정적 용법 vs 계속적 용법 — 콤마 차이

| 형태 | 콤마 | 의미 |
|---|---|---|
| 한정적 | X | 선행사를 한정·구별 |
| 계속적 | O (콤마) | 선행사에 부가 정보 |

**한정적**:
```
The students who studied hard passed the exam.
(공부 열심히 한 학생들만 합격 — 학생을 구별)
```

**계속적** (콤마):
```
The students, who studied hard, passed the exam.
(학생들이 합격 — 그런데 그들은 모두 공부 열심히 했다 — 부가 설명)
```

### 계속적 용법은 that 사용 불가

```
✗ The book, that I read, is interesting.
✓ The book, which I read, is interesting.

✗ This is my brother, that lives in Seoul.
✓ This is my brother, who lives in Seoul.
```

**계속적 = which / who 만**. that 절대 X. 시험 단골.

### that 만 가능 / 더 자연스러운 케이스

| 선행사 | 추천 |
|---|---|
| 최상급 + 명사 | that |
| 서수 + 명사 | that |
| the only / the very / the same | that |
| all / every / no / any + 명사 | that |
| 사람 + 사물 동시 | that |

```
✓ This is the best book that I have ever read. (최상급)
✓ He is the only person that can help. (only)
✓ Look at the boy and the dog that are running. (사람 + 사물)
```

이런 경우 which 보다 that 이 자연스러움.

### 전치사 + 관계대명사

| 패턴 | 사람 | 사물 |
|---|---|---|
| 전치사 앞 (격식) | with whom | in which |
| 전치사 뒤 (구어) | who/that ... with | which/that ... in |

```
✓ The house in which I live is old. (격식)
✓ The house (which/that) I live in is old. (구어)

✗ The house in that I live (전치사 + that X)
```

전치사 + that X 도 시험 단골.

### 소유격 whose / of which

```
✓ The book whose cover is red is mine.
✓ The book of which the cover is red is mine. (격식)
✓ The book the cover of which is red is mine. (격식)
```

세 형태 다 가능. whose 가 가장 자연.

### 관계대명사 생략 — 목적격만

목적격 (whom/which/that) 은 구어에서 자주 생략.

```
✓ The book I read was great. (which/that 생략)
✓ The boy I met is kind. (whom/that 생략)
```

주격은 생략 못 함:

```
✗ The boy is reading is my brother.
✓ The boy who is reading is my brother.
```

단, **주격 + be 동사** 는 함께 생략 가능 (분사 형태로):

```
The man who is sitting → The man sitting (분사)
```

(7-1 단원 분사에서 다룸)

## 대표 문장 (외우기)

> **This is the best book that I have ever read, which my friend recommended.**
>
> 이건 내가 읽어본 최고의 책이다. 친구가 추천한 책이다.

`that I have ever read` (최상급 + that) + `, which my friend recommended` (계속적 — 콤마 + which). 한정 vs 계속.

## 실전 문제 5개

### 문제 1.
어법상 옳은 것은?

① The book, that I read yesterday, is great.
② The book, which I read yesterday, is great.
③ The book whose I read yesterday is great.
④ The book whom I read yesterday is great.

### 문제 2.
다음 빈칸에 알맞은 것은?

`This is the only thing _____ I can do.`

① which  ② that  ③ what  ④ whose

### 문제 3.
다음 중 어법상 **틀린** 문장은?

① The pen which I lost is expensive.
② The pen that I lost is expensive.
③ The pen I lost is expensive.
④ The pen, that I lost, is expensive.

### 문제 4.
빈칸에 들어갈 적절한 것은?

`The house _____ I live is small.`

① in which  ② which in  ③ in that  ④ that in

### 문제 5.
다음 두 문장을 한 문장으로 가장 자연스럽게 합친 것은?

`I bought a car. + The car's color is red.`

① I bought a car which color is red.
② I bought a car whom color is red.
③ I bought a car whose color is red.
④ I bought a car what color is red.

---

## 정답 + 해설

### 1. ② The book, which I read yesterday, is great.

계속적 (콤마) → that X, which 만. ③ 소유격 부적절. ④ whom 사람용.

### 2. ② that

`the only` 다음에는 that 선호.

### 3. ④ The pen, that I lost, is expensive.

계속적 + that X.

### 4. ① in which

전치사 + which (격식). ③ 전치사 + that X.

### 5. ③ I bought a car whose color is red.

소유격 → whose (사람·사물 모두 가능). ① which 의 소유격 X.
