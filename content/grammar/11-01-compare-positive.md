## 핵심 명제

> **as + 형용사/부사 + as = 원급 (같은 정도). 부정은 not as ... as 또는 not so ... as.**

품사 그대로 (원급) 가 들어가는 게 핵심.

---

## 설명

### 기본 형태 — as + 원급 + as

`as + 형용사/부사 (원급) + as` = ~만큼 ~한.

```
✓ Tom is as tall as Jerry. (탐은 제리만큼 키가 크다)
✓ She runs as fast as he does. (그녀는 그만큼 빨리 달린다)
✓ This box is as heavy as that one.
```

원급 = 형용사·부사의 변화 X 형태 (tall, fast, beautiful, quickly).

### 부정 — not as ... as 또는 not so ... as

```
✓ He is not as tall as I (am). (= 그는 나만큼 크지 않다)
✓ She is not so smart as her sister. (= 그녀는 언니만큼 똑똑하지 않다)
```

`not as ... as` 가 더 흔함. `not so ... as` 는 약간 격식.

### 배수 표현 — 두 배 / 세 배 ... + as ... as

`twice / three times / half / one-third + as ... as`

```
✓ He is twice as old as me. (그는 나보다 두 배 나이가 많다)
✓ This bag is three times as expensive as that one.
✓ It cost half as much.
```

배수가 `as ... as` 앞에 옴. 이 패턴 시험 단골.

### as 뒤의 비교 대상 — 주격 vs 목적격

격식 = 주격, 구어 = 목적격.

```
격식: He is as tall as I (am).
구어: He is as tall as me.
```

영어 시험에선 보통 둘 다 인정.

### as + 원급 + a + 명사 + as — 명사도 비교

```
✓ He is as good a singer as his father. (그는 자기 아버지만큼 좋은 가수다)
✓ This is as interesting a book as that one.
```

어순 주의: `as + 형용사 + a/an + 명사`.

### the same ~ as — 같은 ~

`the same + 명사 + as` = 같은 ~.

```
✓ I have the same bag as you.
✓ She gave me the same answer as before.
```

### 자주 시험에 나오는 관용

| 표현 | 의미 |
|---|---|
| as ~ as possible | 가능한 한 ~ |
| as ~ as I can | 내가 할 수 있는 한 ~ |
| as soon as | ~하자마자 |
| as long as | ~하는 한 |
| as far as | ~하는 한 (범위) |
| not so much A as B | A 라기보다는 B |

```
✓ Run as fast as possible. (= as fast as you can)
✓ As long as you study, you will succeed. (~하는 한)
✓ He is not so much a teacher as a friend. (선생님이라기보다는 친구)
```

## 대표 문장 (외우기)

> **He is as tall as me, but he runs twice as fast as I do.**
>
> 그는 나만큼 키가 크다. 그러나 그는 나보다 두 배 빨리 달린다.

`as tall as me` (원급) + `twice as fast as I do` (배수). 두 핵심 패턴.

## 실전 문제 5개

### 문제 1.
다음 빈칸에 알맞은 것은?

`Tom is _____ tall _____ Jerry.`

① more, than  ② as, as  ③ so, that  ④ such, as

### 문제 2.
어법상 옳은 것은?

① He is as taller as me.
② He is as tall as me.
③ He is as tallest as me.
④ He is as more tall as me.

### 문제 3.
다음 중 어법상 **틀린** 문장은?

① This bag is as heavy as that one.
② She is twice as old as me.
③ Run as fast as you can.
④ He is more tall as his brother.

### 문제 4.
빈칸에 들어갈 적절한 것은?

`I have _____ book as you.`

① the same  ② same  ③ a same  ④ similar

### 문제 5.
다음 우리말을 영어로 옮길 때 가장 적절한 것은?

> 그녀는 가능한 한 빨리 왔다.

① She came as fast as possible.
② She came as faster as possible.
③ She came so fast as possible.
④ She came fastest as possible.

---

## 정답 + 해설

### 1. ② as, as

원급 비교 = as ~ as.

### 2. ② He is as tall as me.

원급 = 형용사 원형 (tall). 비교급·최상급 X.

### 3. ④ He is more tall as his brother.

`more tall` 도 X (비교급은 taller 한 단어), `as` 도 X (비교급은 than). → `He is taller than his brother`.

### 4. ① the same

the same + 명사 + as.

### 5. ① She came as fast as possible.

as ~ as possible (관용).
