## 핵심 명제

> **as if + 가정법 과거 = 마치 ~인 것처럼 (실제 X). as if + had p.p. = 마치 ~였던 것처럼.**

I wish 와 같은 시제 룰.

---

## 설명

### as if + 가정법 과거 (현재 사실 반대)

마치 ~인 것처럼 (실제는 그렇지 않음).

```
✓ He talks as if he were a doctor. (그는 의사가 아닌데, 의사인 것처럼 말한다)
✓ She acts as if she knew everything. (사실은 모르는데)
✓ He looks as if he were sick. (안 아픈데, 아픈 것처럼)
```

### as if + 가정법 과거완료 (과거 사실 반대)

마치 ~였던 것처럼 (실제는 그렇지 않았음).

```
✓ He talks as if he had been rich. (부자였던 적 없는데)
✓ She acts as if she had won the prize. (안 받았는데)
```

### 시제 표

| 형태 | 의미 |
|---|---|
| as if + 과거 | 마치 ~인 것처럼 (현재 반대) |
| as if + had p.p. | 마치 ~였던 것처럼 (과거 반대) |

### as if + 직설법 — 사실일 가능성 있음

화자가 실제로 그럴 수도 있다고 생각할 때는 직설법.

```
✓ He looks as if he is sick. (직설법 — 실제 아픈 것 같다)
✓ It looks as if it will rain. (비올 것 같다)
```

같은 as if 인데 시제로 의미 다름:
- 가정법 과거: 사실 반대
- 직설법: 사실일 가능성

### as though = as if (완전 동의어)

```
✓ He talks as though he were a king.
✓ She acts as though she had seen a ghost.
```

격식·문어체에서 as though 가 선호되기도 함.

### like — 구어 (격식 X)

`like` 도 같은 의미로 쓰임 (구어).

```
✓ He acts like he were the boss. (구어, 비표준)
✓ She talks like she knew everything. (구어)
```

격식 글에선 as if / as though 사용.

### 예시 비교

```
✓ He drinks as if there were no tomorrow. (마치 내일이 없는 것처럼 — 실제 있음)
✓ She runs as if she were a horse. (말이 아닌데)
✓ He talks as if he had met the president. (만난 적 없는데)
```

---

## 대표 문장 (외우기)

> **He talks as if he were a doctor, but he acts as if he had never studied medicine.**
>
> 그는 의사인 것처럼 말하지만, 의학을 공부한 적 없는 것처럼 행동한다.

`as if were` (현재 반대) + `as if had + p.p.` (과거 반대). 두 시제.

---

## 실전 문제 5개

### 문제 1.
다음 빈칸에 알맞은 것은?

`He talks as if he _____ a millionaire.`

① is  ② were  ③ has been  ④ will be

### 문제 2.
어법상 옳은 것은?

① She acts as if she knows everything, but she doesn't.
② She acts as if she knew everything, but she doesn't.
③ She acts as if she has known everything, but she doesn't.
④ She acts as if she will know everything, but she doesn't.

### 문제 3.
다음 두 문장의 의미 차이로 알맞은 것은?

`A: He looks as if he is sick.`
`B: He looks as if he were sick.`

① 둘 다 안 아픔
② A 는 아플지도 모름, B 는 안 아픔
③ A 는 안 아픔, B 는 아픔
④ 둘 다 아픔

### 문제 4.
다음 중 어법상 **틀린** 문장은?

① He talks as if he had been to Paris.
② She walks as though she were a model.
③ He talks as if he is a king. (사실 아님)
④ It looks as if it will rain.

### 문제 5.
다음 우리말을 영어로 옮길 때 가장 적절한 것은?

> 그녀는 마치 유령을 본 것처럼 비명을 질렀다 (사실 본 적 없음).

① She screamed as if she sees a ghost.
② She screamed as if she saw a ghost.
③ She screamed as if she had seen a ghost.
④ She screamed as if she will see a ghost.

---

## 정답 + 해설

### 1. ② were

as if + 가정법 과거 = 사실 반대 (백만장자 아님).

### 2. ② She acts as if she knew everything, but she doesn't.

뒷 절 "but she doesn't" 가 사실 반대 명시 → 가정법 과거.

### 3. ② A 는 아플지도 모름, B 는 안 아픔

A 직설법 = 사실일 가능성. B 가정법 = 사실 반대.

### 4. ③ He talks as if he is a king. (사실 아님)

사실 아니라면 가정법 과거 → `as if he were a king`.

### 5. ③ She screamed as if she had seen a ghost.

비명 (과거) 시점에 더 이전의 가정 (유령 봄) → 가정법 과거완료.
