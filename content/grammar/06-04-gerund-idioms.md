## 핵심 명제

> **be busy / be worth / can't help / feel like / spend 시간 + V-ing. 패턴을 통째로 외워라.**

자주 출제되는 10개 관용 패턴.

---

## 설명

동명사 (-ing) 가 들어가는 고정 패턴. 통째로 외우는 게 답.

### 10대 핵심 패턴

| 패턴 | 의미 | 예 |
|---|---|---|
| **be busy + V-ing** | ~하느라 바쁘다 | I'm busy studying. |
| **be worth + V-ing** | ~할 만하다 | This book is worth reading. |
| **can't help + V-ing** | ~하지 않을 수 없다 | I can't help laughing. |
| **feel like + V-ing** | ~하고 싶다 | I feel like eating pizza. |
| **spend 시간 + V-ing** | ~하는 데 시간을 쓰다 | I spent an hour reading. |
| **have difficulty (in) + V-ing** | ~하는 데 어려움 있다 | I have difficulty learning Chinese. |
| **It is no use + V-ing** | ~해봐야 소용없다 | It's no use crying. |
| **There is no + V-ing** | ~할 수 없다 | There is no telling. |
| **on/upon + V-ing** | ~하자마자 | On seeing me, she ran. |
| **be used to + V-ing** | ~에 익숙하다 | I'm used to driving. |

### be used to vs used to — 단원 4-5 의 함정 재확인

| 표현 | 의미 |
|---|---|
| **used to + 원형** | 과거 습관 (지금은 X) |
| **be used to + V-ing** | ~에 익숙해진 |
| **get used to + V-ing** | ~에 익숙해지다 |

```
✓ I used to live in Seoul. (예전 살았음)
✓ I am used to living in Seoul. (사는 데 익숙)
✓ I am getting used to spicy food. (익숙해지는 중)
```

가장 헷갈리는 함정. **be used to 뒤는 -ing/명사** (to V 원형 X).

### 동사 + 전치사 + V-ing

전치사 뒤는 무조건 동명사 (to V 못 옴).

| 패턴 | 예 |
|---|---|
| look forward to + V-ing | ~을 기대하다 |
| be interested in + V-ing | ~에 관심 있다 |
| be afraid of + V-ing | ~을 두려워하다 |
| object to + V-ing | ~에 반대하다 |
| confess to + V-ing | ~을 자백하다 |
| think of + V-ing | ~을 생각하다 |
| insist on + V-ing | ~을 주장하다 |
| succeed in + V-ing | ~에 성공하다 |
| dream of + V-ing | ~을 꿈꾸다 |
| prevent A from + V-ing | A 가 ~하지 못하게 |

`look forward to` 의 to 는 **전치사** (부정사 X). 따라서 to + V-ing.

```
✗ I look forward to see you.
✓ I look forward to seeing you. (시험 단골)
```

### what about / how about + V-ing — 제안

```
✓ What about going to the movies?
✓ How about having lunch together?
```

### no + 동명사 — 금지 표지

```
✓ No smoking. (금연)
✓ No parking.
```

### so + 형용사 + as to V vs so as to V

부정사 패턴인데 동명사 패턴과 헷갈림 주의.

```
✓ He was so kind as to help me. (그렇게 친절해서 도와줬다)
✓ He worked hard so as to succeed. (성공하기 위해 — 목적)
```

## 대표 문장 (외우기)

> **I look forward to seeing you. I'm busy preparing the meeting.**
>
> 만나뵙길 기대합니다. 회의 준비하느라 바쁩니다.

`look forward to seeing` (전치사 to + -ing — 시험 단골) + `busy preparing` (be busy + -ing). 두 핵심 패턴.

## 실전 문제 5개

### 문제 1.
어법상 옳은 것은?

① I look forward to see you again.
② I look forward to seeing you again.
③ I look forward to saw you again.
④ I look forward see you again.

### 문제 2.
다음 빈칸에 알맞은 것은?

`I am used to _____ early.`

① wake up  ② to wake up  ③ waking up  ④ woke up

### 문제 3.
다음 중 어법상 **틀린** 문장은?

① I can't help laughing at his joke.
② This movie is worth watching.
③ He spent two hours to play games.
④ It's no use crying over spilt milk.

### 문제 4.
빈칸에 들어갈 적절한 것은?

`On _____ the news, she screamed.`

① to hear  ② hearing  ③ heard  ④ hear

### 문제 5.
다음 우리말을 영어로 옮길 때 가장 적절한 것은?

> 그는 매운 음식 먹는 데 익숙해졌다.

① He used to eat spicy food.
② He is used to eat spicy food.
③ He is used to eating spicy food.
④ He used to eating spicy food.

---

## 정답 + 해설

### 1. ② I look forward to seeing you again.

look forward to + V-ing (to 가 전치사). 가장 자주 틀리는 패턴.

### 2. ③ waking up

be used to + V-ing/명사. ① ② to V 부적절. ④ 과거 X.

### 3. ③ He spent two hours to play games.

spend 시간 + V-ing → `playing games`. 나머지 정상.

### 4. ② hearing

on + V-ing = ~하자마자. ① to V X. ③ ④ 동사 형태 X.

### 5. ③ He is used to eating spicy food.

be used to + V-ing = 익숙. ① 과거 습관 (지금은 X) — 의미 정반대. ② to V X. ④ used to + V-ing 형태 X.
