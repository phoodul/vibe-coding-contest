## 핵심 명제

> **명사 앞에서 명사를 한정하는 단어. each/every 는 단수, both/all 은 복수.**

한정사 단복수 매칭이 시험 단골.

---

## 설명

### 한정사란

명사 앞에 놓여 명사의 범위·수량·소유 등을 한정.

| 종류 | 예 |
|---|---|
| 관사 | a / an / the |
| 지시 | this / these / that / those |
| 소유격 | my / your / his / her / its / our / their |
| 부정 | some / any / no |
| 수량 | many / much / few / little / all / both / each / every |

### 지시 한정사 — 단복수 매칭

| 거리·수 | 단수 | 복수 |
|---|---|---|
| 가까운 | this | these |
| 먼 | that | those |

```
✓ This book is mine.
✓ These books are mine.
✓ That dog is friendly.
✓ Those dogs are friendly.
```

### each / every — 단수 + 단수 동사

| 형태 | 사용 |
|---|---|
| each + 단수 | 각각의 |
| every + 단수 | 모든 (3개 이상) |
| each of + 복수 | ~의 각각 |

```
✓ Each student has a book. (each + 단수 + 단수동사)
✓ Every man is mortal.
✓ Each of them has a book.

✗ Each students have a book.
✓ Each student has a book.
```

`every` 는 every two days (이틀마다) 같은 패턴 외엔 단수.

### both — 복수 + 복수

`both` = 둘 다.

```
✓ Both students are smart. (both + 복수)
✓ Both of them are correct.
```

### all — 단수·복수 가능

`all + 단수`(전체) / `all + 복수`(전부).

```
✓ All the food is gone. (음식 단수, 전체 양)
✓ All the students are here. (학생들 복수)
```

### each vs every 미묘한 차이

| 단어 | 어감 |
|---|---|
| each | 개별 강조 (한 명 한 명) |
| every | 전체 강조 (예외 없이) |

```
Each student answered differently. (각자 다르게 — 개별)
Every student must wear a uniform. (모두 — 예외 없이)
```

### one / another / the other / others

선택지가 여럿일 때 지칭.

| 형태 | 의미 |
|---|---|
| one | 하나 (셋 이상 중) |
| another | 또 다른 하나 |
| the other | 둘 중 나머지 하나 |
| others | 다른 것들 (복수) |
| the others | 나머지 (복수) |

```
✓ I have two cats. One is white, the other is black.
(둘 중 하나/나머지 하나)

✓ I have three cats. One is white, another is black, the other is gray.
(셋 중 하나/또 다른 하나/마지막 하나)

✓ Some students study, others play.
(일부는 ~, 다른 일부는 ~)
```

### no — 한정사 부정

`no + 단수/복수/불가산` = 어떤 ~도 아님.

```
✓ No student is here. (단수)
✓ No students are here. (복수)
✓ No money is needed. (불가산)
```

### 한정사 vs 대명사 — 형태 같음

같은 단어가 한정사·대명사 둘 다 됨.

```
✓ This book is mine. (한정사 — book 앞)
✓ This is my book. (대명사 — 단독)
```

## 대표 문장 (외우기)

> **Each student has a book, and both teachers are kind.**
>
> 각 학생은 책을 한 권씩 가지고 있고, 두 선생님 모두 친절하다.

`Each + 단수` + `both + 복수`. 단복수 매칭의 핵심.

---

## 실전 문제 5개

### 문제 1.
어법상 옳은 것은?

① Each students have a book.
② Each student have a book.
③ Each student has a book.
④ Each students has a book.

### 문제 2.
다음 빈칸에 알맞은 것은?

`_____ students are kind.`

① Both  ② Each  ③ Every  ④ Either

### 문제 3.
다음 중 어법상 **틀린** 문장은?

① All students are present.
② Every man is mortal.
③ Both of them is correct.
④ Each of them has a book.

### 문제 4.
빈칸에 들어갈 적절한 것은?

`I have two cats. One is white, _____ is black.`

① another  ② the other  ③ other  ④ others

### 문제 5.
다음 우리말을 영어로 옮길 때 가장 적절한 것은?

> 모든 학생이 시험에 합격했다.

① Every students passed the exam.
② Every student passes the exam.
③ Every student passed the exam.
④ Every students has passed the exam.

---

## 정답 + 해설

### 1. ③ Each student has a book.

each + 단수 + 단수동사 has.

### 2. ① Both

`students` 복수 → both 가능. ② each 단수, ③ every 단수, ④ either 단수.

### 3. ③ Both of them is correct.

both 복수 → are correct.

### 4. ② the other

둘 중 하나 (white) + 나머지 하나 (the other). ① another 는 셋 이상에서.

### 5. ③ Every student passed the exam.

every + 단수 + 단수동사. 시제는 과거 (passed).
