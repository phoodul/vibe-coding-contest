# Crisis Intervention 종합 구현 계획

## 목표
crisis_intervention.md의 **7개 섹션 전체**를 활용하여, 위기 학생이 즉시 도움받을 수 있는 종합 자원 페이지를 구현한다.

## 구현 구조: FAB 버튼 + 전용 `/crisis` 페이지

### A. CrisisButton (FAB) — `src/components/shared/crisis-button.tsx`
- 대시보드 우하단 고정
- 클릭 → `/crisis` 페이지로 이동
- pulse 애니메이션 (과하지 않게)

### B. `/crisis` 페이지 — `src/app/crisis/page.tsx`

#### 섹션 1: 히어로 — "혼자가 아닙니다"
- 따뜻한 메시지 + 긴급 2개(119 응급, 112 경찰) 즉시 전화 카드
- `tel:` 링크, 빨간 강조

#### 섹션 2: "지금 어떤 상황인가요?" — 상황별 행동 가이드
원문 §6의 3가지 상황을 **인터랙티브 카드**로 구현:

| 카드 | 펼치면 보이는 내용 (원문 기반) |
|------|------|
| "자살·자해 생각이 있어요" | ① 믿을 수 있는 어른에게 지금 기분 말하기 ② 안전한 장소로 이동 ③ 도구 멀리 치우기 ④ 109/1577-0199/1388 전화 ⑤ 위험하면 119 신고·응급실 |
| "학교폭력을 당하고 있어요" | ① 혼자가 아님을 기억 ② 부모/담임/학폭 담당 선생님에 알리기 ③ 증거 확보(캡처 등) ④ 117 신고 ⑤ 심하면 112 신고 |
| "우울하거나 힘들어요" | ① 주변에 고민 털어놓기 ② 1388 청소년 전화 ③ 학교 Wee센터 상담 ④ 지역 정신건강복지센터 방문 ⑤ 자가진단 앱 활용 |

#### 섹션 3: 24시간 상담 자원 전체 디렉토리
원문 §1 + 표 전체를 **카드 그리드**로:

| 서비스명 | 번호 | 운영 | 대상 | 접근방법 | 핵심설명 |
|---------|------|------|------|---------|---------|
| 자살예방상담전화 (보건복지부) | 109 | 24h | 전 연령 | 전화/카카오톡/앱/문자 | 자살위험 전문상담, 위기개입 |
| 정신건강위기상담전화 (보건복지부) | 1577-0199 | 24h | 전 연령 | 전화 | 중증 우울·자해충동·급격한 위기 |
| 청소년전화 (한국청소년상담복지개발원) | 1388 | 24h | 만9~24세/보호자 | 전화/문자/카카오/웹/챗봇/심리검사 | 학교·가정·자살충동 등 청소년 전용 |
| 생명의전화 (한국생명의전화) | 1588-9191 | 24h | 전 연령 | 전화/카카오 '채널 라임' | 자살위기·외로움 상담, SOS 현장전화 운영 |
| 학교폭력신고 | 117 | 24h | 전 연령 | 전화 | 학교폭력 신고·상담, 교육청 조치 요청 |
| 응급의료 | 119 | 24h | 전 연령 | 전화 | 생명 위급 시 응급실 이송 |
| 경찰 | 112 | 24h | 전 연령 | 전화 | 강력범죄·타인 피해 위급 상황 |
| "다 들어줄 개" (교육부) | 1661-5004 | 24h | 만9~24세 | 카카오톡/앱/문자 | 익명 SNS 상담, 전문상담원 연결 |
| 여성긴급전화 | 1366 | 24h | 전 연령 | 전화 | 가정폭력·성폭력 위기 상담 |

각 카드: 번호 `tel:` 링크 + 접근방법 배지 + 운영시간 + 대상 태그

#### 섹션 4: 학교 내 지원 체계
원문 §2 기반:
- **Wee 프로젝트**: Wee 클래스(학교 내) → Wee 센터(교육지원청) → 중앙지원센터(044-415-2157)
- **상담교사 면담**: 학교에 요청 가능, 평일 근무시간
- **위기 대응 절차**: 자해 발견 시 교사 초기 대응 → 보호자 통보 → 전문기관 연계
- **신고 의무**: 교원은 아동학대·가정폭력·학교폭력·성범죄 신고의무자

#### 섹션 5: 의료·응급 지원
원문 §3 기반:
- **생명사랑위기대응센터**: 전국 75개 병원 설치, 집중치료 + 재발 방지 프로그램
- **응급 지원비**: 교육청에서 응급 입원·진료비 일부 보조 가능
- **정신건강의학과**: 종합병원 정신건강센터, 지역 정신건강복지센터, 외래 상담·약물·심리치료

#### 섹션 6: 온라인·디지털 자원
원문 §4 + §5 기반:
- **앱**: "다 들어줄 개", "마음 콕", "디지털 마음정원" — 심리검진/자가진단
- **웹**: 정신건강정보 포털, "마음검진" 사이트(우울·불안·자기파괴 위험도 측정)
- **익명 상담**: 1:1 사이버상담 게시판(비공개), 카카오톡 오픈채팅
- **비밀보장 원칙**: 별도 동의 없이 개인정보 수집 안 함, 신원·내용 비밀 보장. 단 생명 위협 시 응급조치 가능

#### 섹션 7: 지역별 자원 (아코디언)
원문 §7 기반 — 도시별 접기/펼치기:

| 지역 | 특화 자원 |
|------|---------|
| 서울 | 서울학생통합콜센터(24h), 마음치유학교, 스쿨라인(1577-7018, 교사용, 평일 9-18시), 전문상담교사 전교 배치 추진 |
| 부산 | 부산자살예방센터, 부산멘탈프렌즈·버디 프로그램, 부산시정신건강센터 |
| 대구 | 대구자살예방센터, 대구청년마음건강센터 |
| 인천 | 인천정신건강복지센터, 구·군 청소년상담센터, 마음아카데미 |
| 광주·대전·울산 등 | 각 시도 정신건강복지센터(1577-0199·109), 지역맞춤 상담·지원 프로그램 |

#### 섹션 8: 위기대응 흐름도
원문 §6의 mermaid 플로차트를 **시각적 스텝 UI**로 구현:
```
위험하다고 느끼나요?
├─ 예, 긴급 → 119/112 신고 → 응급치료 → 안정화 후 지속 케어
└─ 아니오 → 상담전화(109/1577/1388) 
           → 학교상담(Wee/부모)
           → 지역센터·의료기관
           → 지속 케어
```
