# Mind Palace 비주얼 재설계 — 대형 이미지 + Hotspot 방식

## 핵심 변경

**기존**: 소품 45개 (60~160px) × 1키워드 → 평면, 공중 부유, 겹침
**변경**: 대형 이미지 6~8개 × 다수 hotspot → 입체적, 공간 밀착, 고밀도

## 컨셉: 호랑이 원리

하나의 대형 이미지(예: 호랑이 그림) 안에 부위별 hotspot:
- 호랑이 눈 → "관찰/통찰" 관련 내용
- 호랑이 입 → "표현/주장" 관련 내용
- 호랑이 앞다리 → "실천/행동" 관련 내용
- 호랑이 배 → "핵심/본질" 관련 내용
- 호랑이 꼬리 → "결론/마무리" 관련 내용

→ 1 이미지에 5~8개 내용 매핑 가능

## 근정전 공간 배치 (ch3 "사회와 윤리")

```

│  뒷벽:  │   일월오봉도 (대형)    │                │

│         └───────────────────────┘                │
│                                                  │

│   배/등/꼬리)       │        다리/꼬리)          │
│  → s2 사회정의      │       → s3 국가시민윤리    │
│                     │                            │
├─────────────────────────────────────────────────┤
│  바닥:                                           │
│  ┌─────────────────────────────────────┐        │
│  │         궁중 카펫 (대형)             │        │
│  │  중앙문양/테두리/모서리4개/줄무늬    │        │

│                                                  │
│  좌측바닥: 어좌 (대형, 입체)                     │

│  우측바닥: 향로 + 촛대 세트                      │

└─────────────────────────────────────────────────┘
```

## 이미지별 상세 hotspot 매핑

### 1. 일월오봉도 (뒷벽 중앙, ~40% 너비)
| 부위 | 내용 | 섹션 |
|------|------|------|
| 해(태양) | 직업의 의미 — 자아실현 | s1-c1 |
| 달 | 맹자 — 노심·노력 분업 | s1-c1 |
| 봉우리 5개 | 칼뱅 소명설 / 마르크스 소외 / 순자 예 / 의사윤리 / 변호사윤리 | s1-c1~c2 |
| 소나무 2그루 | 프리드먼(주주자본주의) / 프리먼(이해관계자) | s1-c2 |
| 파도 | 교사윤리 — 공공이익 > 사적이익 | s1-c2 |

## 데이터 구조 변경

```typescript
// 기존: 1 structure = 1 visualType + bullets[]
interface RoomStructure {
  id: string;
  keyword: string;
  position: { x: number; y: number };
  visualType: VisualType; // "throne" | "scroll" | ...
  bullets: Bullet[];
}

// 변경: 1 sceneImage = 1 대형 이미지 + hotspots[]
interface SceneImage {
  id: string;
  name: string;              // "일월오봉도", "호랑이 벽화"
  placement: "back-wall" | "left-wall" | "right-wall" | "floor" | "ceiling";
  position: { x: number; y: number };   // % 기준 공간 내 위치
  size: { w: number; h: number };        // % 기준 크기 (예: 40x30)
  hotspots: Hotspot[];
}

interface Hotspot {
  id: string;
  label: string;             // "호랑이 눈", "해(태양)"
  region: { x: number; y: number; w: number; h: number }; // 이미지 내 상대 위치 %
  keyword: string;
  text: string;
  section: string;           // "s1-c1", "s2-c3" 등
}
```

## 렌더링 변경

1. **대형 SVG/CSS 아트**: 각 SceneImage를 200~500px 크기의 상세한 SVG로 렌더링
2. **Hotspot 오버레이**: 이미지 위에 투명한 클릭 영역, hover 시 해당 부위 하이라이트
3. **말풍선**: hotspot 클릭 시 해당 부위 옆에 연결선 + 글래스 말풍선
4. **접지감**: 바닥 오브젝트에 그림자, 벽 오브젝트에 벽면 밀착 transform
5. **카펫**: perspective transform으로 바닥에 깔린 느낌

## 검증

1. `/palace/demo` 접속 → 6~8개 대형 이미지가 공간에 밀착 배치
2. 각 이미지 내 hotspot hover → 부위 하이라이트
3. hotspot 클릭 → 연결선 + 말풍선
4. 나레이터 모드 → 이미지별 → hotspot 순서로 자동 진행
5. `npm run build` 통과
