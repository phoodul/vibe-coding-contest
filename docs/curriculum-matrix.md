## 자체 제작 워크플로우 (헤밍웨이 v2 패턴 일반화)

각 과목 신설 시:

```
1. SubjectKey 확장 — src/lib/mind-map/build-tree.ts
   export type SubjectKey =
     | "ethics" | "biology" | "korean"
     | "<new-key>";  // ← 추가

2. 단원·chapter 목차 작성 — docs/curriculum-<subject>.md
   (헤밍웨이의 docs/grammar-curriculum.md 패턴)

3. chapter MDX 또는 TS 작성 — src/lib/data/textbooks/<subject>-ch{N}.ts
   (또는 헤밍웨이 패턴: content/<subject>/<chapter-slug>.md)

4. structured/ 노드 — src/lib/data/textbooks/structured/<subject>-structured-ch{N}.ts
   (마인드맵 트리용)

5. SUBJECT_DATA Record — build-tree.ts 에 textbook + structured 등록

6. 마인드맵 페이지 SUBJECTS 배열 — src/app/mind-map/page.tsx 에 추가
   { key: "<new-key>", label: "<표시명>", icon: "<emoji>" }

7. 소크라테스 튜터 진입점 — /tutor 페이지에서 SubjectKey 선택 UI
   (이미 SUBJECTS 가 있음 — `src/lib/ai/tutor-prompt.ts` SUBJECTS 와 통합)
```
