# PRD (Product Requirements Document)

> 작성일: 2026-04-06 | 작성자: 정성수 | 버전: v1.0

## 1. 프로젝트 개요

### 프로젝트명
**MindPalace (마인드팰리스)** — AI 기억의 궁전 학습 + 교사 AI 어시스턴트

### 한 줄 설명
교과서를 마인드맵으로 펼치고, 기억의 궁전에 저장하라

### 해결하려는 문제

**학생 측:**
- 교과서 단원의 핵심 개념을 체계적으로 정리하고 기억하는 도구가 없음
- 단순 반복 암기 → 망각 곡선에 의해 빠르게 소실
- 개인화된 학습 전략 부재 (모두 같은 방식으로 암기)

**교사 측:**
- 수행평가 200-400명 개별 피드백 불가능 (시간 부족)
- 생활기록부 작성에 막대한 시간 소요
- 학생 비교과 활동을 종합하여 문장으로 만드는 작업이 기계적이지만 시간 집약적

**왜 지금인가:**
- 2025년 수행평가 확대 정책 → 교사 부담 급증
- AI 디지털교과서 도입 시작 → AI 교육 도구 수용도 최고조
- 기억법(Method of Loci)은 과학적으로 검증된 최고의 암기 기법이지만, 디지털 도구화된 적 없음

### 핵심 가치 제안 (Value Proposition)
> "학생이 교과서 단원을 선택하면, AI가 마인드맵을 생성하고 기억의 궁전에 배치하여 장기 기억을 형성한다. 교사는 AI가 생성한 피드백 초안으로 200명의 학생에게 개별 피드백을 제공한다."

## 3. 핵심 기능 (Scope)

### Must-Have (Day 2~5에 완성)

1. **기억의 궁전 — 마인드맵 생성** 
   학년/과목/단원 선택 → AI가 성취기준 기반 마인드맵(핵심 개념, 관계, 하위 주제) 생성 → 인터랙티브 시각화

2. **기억의 궁전 — 장소 선택 & 배치**
   10-15개 사전 정의 장소(한옥, 도서관, 박물관 등) 중 선택 → 장소 구역별 마인드맵 노드 배치 → AI가 기억 연결 전략 제안

3. **기억의 궁전 — 복습 모드**
   장소를 "걸어다니며" 배치된 개념을 회상하는 인터랙티브 복습

4. **교사 — 수행평가 피드백 생성**
   루브릭(평가기준) 입력 + 학생 답안 입력 → AI가 소크라틱 피드백 초안 생성 → 교사 검토/수정/승인

5. **교사 — 생활기록부 초안**
   학생 데이터(출결, 수행평가, 동아리, 봉사활동 등) 입력 → AI가 생기부 문장 생성

### Nice-to-Have (시간 되면)
1. 복습 알림 (망각 곡선 기반 스케줄링)
2. 학습 분석 대시보드 (어떤 단원을 언제 학습했는지)

### Out of Scope (발표에서 로드맵으로 언급)
1. 교권 침해 신고 시스템
2. 교과서 기반 자동 해설서
3. 학부모 리포트
4. 모바일 앱

## 4. Wow Feature (차별화 포인트)

> 심사위원이 기억할 **한 장면**을 정의한다.

- **무엇:** 교과서 단원을 선택하면 마인드맵이 생성되고, 한옥의 대청마루/사랑방/부엌 등에 개념이 하나씩 배치되는 인터랙티브 경험
- **왜 Wow인가:** 기억의 궁전은 2,500년 역사의 가장 강력한 기억법이지만, 디지털 도구로 구현한 사례가 전무. AI + 장소법의 세계 최초 결합.
- **시연 시나리오:** "생명과학Ⅰ 세포와 세포분열을 선택합니다 → AI가 핵심 개념(세포막, 핵, 미토콘드리아, 체세포분열, 감수분열 등)의 마인드맵을 생성합니다 → 경복궁을 기억의 궁전으로 선택합니다 → 근정전에 '세포핵(궁전의 중심, 모든 명령이 나오는 곳)', 경회루에 '미토콘드리아(연못 위 에너지가 넘치는 누각)' 가 배치됩니다 → 복습 모드에서 경복궁을 걸어다니며 개념을 떠올립니다"

## 5. 사용자 시나리오 (User Flow)

### 시나리오 1: 학생 — 기억의 궁전으로 학습
1. 학생이 회원가입/로그인 (학생 역할 선택)
2. 대시보드에서 "새 학습 시작" 클릭
3. 과목 → 단원 선택 (예: 생명과학Ⅰ > 세포와 세포분열)
4. AI가 마인드맵 생성 (스트리밍으로 노드가 하나씩 나타남)
5. "기억의 궁전 만들기" 클릭
6. 장소 카드 중 하나 선택 (예: 경복궁)
7. 장소의 구역 맵이 표시되고, 마인드맵 노드가 각 구역에 배치됨
8. AI가 각 배치에 대한 기억 연결 스토리를 생성 ("근정전의 웅장한 기둥처럼, 일차함수는 수학의 기둥입니다...")
9. "복습 시작" → 장소를 순서대로 방문하며 빈칸 채우기

### 시나리오 2: 교사 — 수행평가 피드백 생성
1. 교사가 로그인 (교사 역할 선택)
2. "수행평가 피드백" 메뉴 선택
3. 평가 제목 + 루브릭(채점 기준) 입력
4. 학생 답안 텍스트 붙여넣기
5. AI가 루브릭 기반 피드백 초안 생성 (스트리밍)
6. 교사가 수정 후 "승인" → 저장

### 시나리오 3: 교사 — 생활기록부 초안
1. 교사가 "생기부 작성" 메뉴 선택
2. 학생 정보 입력: 이름, 출결, 수행평가 점수/내용, 동아리, 봉사활동, 특기사항
3. AI가 생기부 스타일의 문장 생성
4. 교사가 수정 후 저장/복사

## 6. 기술 설계 (개요)

### 페이지 구조
```
/ (랜딩 — 벤토 그리드 + 글래스모피즘)
├── /login
├── /signup
├── /dashboard (역할별 분기)
├── /palace (기억의 궁전 메인)
│   ├── /palace/new (단원 선택 → 마인드맵 생성)
│   ├── /palace/[id] (궁전 상세 — 장소 + 노드 배치)
│   └── /palace/[id]/review (복습 모드)
├── /teacher (교사 전용)
│   ├── /teacher/feedback (수행평가 피드백)
│   └── /teacher/record (생기부 초안)
└── /settings
```

### DB 테이블 (Supabase)
| 테이블명 | 주요 컬럼 | 용도 |
|----------|----------|------|
| profiles | id, role(student/teacher), display_name | 사용자 프로필 + 역할 |
| mind_maps | id, user_id, subject, grade, unit, nodes(jsonb) | 생성된 마인드맵 |
| palaces | id, user_id, mind_map_id, location_key, placements(jsonb) | 기억의 궁전 배치 |
| feedback_drafts | id, teacher_id, title, rubric, student_answer, ai_feedback, edited_feedback | 수행평가 피드백 |
| student_records | id, teacher_id, student_name, data(jsonb), ai_draft, edited_draft | 생기부 초안 |

### API 엔드포인트
| Method | Path | 설명 |
|--------|------|------|
| POST | /api/mindmap/generate | 단원 → AI 마인드맵 생성 (스트리밍) |
| POST | /api/palace/connect | 마인드맵 노드 + 장소 → 기억 연결 스토리 생성 |
| POST | /api/teacher/feedback | 루브릭 + 답안 → 피드백 초안 생성 |
| POST | /api/teacher/record | 학생 데이터 → 생기부 초안 생성 |

### AI 에이전트 설계

**마인드맵 생성 에이전트:**
- 입력: 학년, 과목, 단원명, 성취기준
- 출력: JSON 구조의 마인드맵 (중심 개념 + 하위 노드 + 관계)
- 패턴: streamObject (Vercel AI SDK)

**기억 연결 에이전트:**
- 입력: 마인드맵 노드 + 장소 구역 정보
- 출력: 각 노드-장소 연결에 대한 기억 스토리
- 패턴: streamText

**교사 피드백 에이전트:**
- 입력: 루브릭 + 학생 답안
- 출력: 소크라틱 방식 피드백 초안
- 패턴: streamText

**생기부 에이전트:**
- 입력: 학생 활동 데이터
- 출력: 생활기록부 스타일 문장
- 패턴: streamText

## 9. 교육과정 데이터 (데모용)

### 데모 범위
- **국어 — 언어와 매체**: 음운 체계, 단어의 형성, 매체 언어의 특성 (3단원)
- **사회 — 생활과 윤리**: 생명윤리, 사회윤리, 과학기술 윤리 (3단원)
- **과학 — 생명과학Ⅰ**: 세포와 세포분열, 유전, 생태계와 상호작용 (3단원)
- 성취기준은 2022 개정 교육과정 공공 데이터 활용 (저작권 무관)

### 장소 목록 (10개)
1. 경복궁 — 근정전, 경회루, 향원정, 교태전, 사정전
2. 한옥마을 — 대청마루, 사랑방, 안방, 부엌, 마당
3. 국립중앙박물관 — 선사실, 고려실, 조선실, 기획전시실, 야외정원
4. 서울도서관 — 1층 로비, 열람실, 서고, 옥상정원, 계단홀
5. 남산타워 — 전망대, 사랑의 자물쇠, 케이블카, 팔각정, 계단
6. 광화문광장 — 세종대왕상, 이순신장군상, 분수대, 해치상, 육조거리
7. 창덕궁 후원 — 부용정, 연경당, 옥류천, 존덕정, 관람정
8. 제주 돌하르방공원 — 입구, 돌담길, 감귤밭, 전망대, 해안절벽
9. 불국사 — 청운교, 다보탑, 석가탑, 대웅전, 극락전
10. 수원화성 — 장안문, 화홍문, 방화수류정, 서장대, 공심돈
