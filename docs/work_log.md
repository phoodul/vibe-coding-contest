## C. AI 모델 사용 현황

### C-1. API별 AI 모델 및 시스템 프롬프트 요약

#### `/api/tutor` — 소크라테스 AI 튜터
- **모델:** `claude-sonnet-4-20250514` (Claude Sonnet 4)
- **SDK:** `@ai-sdk/anthropic` + `ai` (Vercel AI SDK) `streamText`
- **시스템 프롬프트 요약:**
  - 소크라테스식 Guided Learning (하버드 케이스 메서드)
  - AI가 먼저 질문, 절대 답을 직접 주지 않음
  - 4단계 수업 진행: 개관 질문 → 심화 탐구 → 연결 짓기 → 정리 & 확장
  - 힌트 레벨 시스템 (Lv1~Lv3)
  - 5개 교과 지원: 국어(언어와 매체), 사회(생활과 윤리), 과학(생명과학I), 수학(수학I), 영어(영어I)
  - 각 교과 4~5개 단원, 단원별 핵심 개념 + 오프너 질문 포함

#### `/api/conversation` — AI 영어 회화 파트너
- **모델 (회화):** `claude-haiku-4-5-20251001` (Claude Haiku 4.5) — 빠른 응답 속도
- **모델 (리포트):** `claude-sonnet-4-20250514` (Claude Sonnet 4)
- **SDK:** `@ai-sdk/anthropic` + `ai` `streamText`
- **시스템 프롬프트 요약:**
  - 한국 학생 영어 회화 파트너
