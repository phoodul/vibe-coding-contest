## 본문

안녕하세요 베타 테스터님,

기존 Euler Tutor가 **Legend Tutor**로 새롭게 태어났습니다. 이제 다음 5명의 수학 거장이 학생의 수준과 영역에 맞춰 라우팅됩니다:

- **라마누잔** — 계산의 달인 (단순/중등 난이도 즉시 답변)
- **가우스** — 미적분·공통의 깊은 통찰 (Gemini 3.1 Pro)
- **폰 노이만** — 기하·해석의 거장 (GPT-5.5)
- **오일러** — 모든 수학의 다리 (Opus 4.7 agentic)
- **라이프니츠** — 미적분의 창시자 (Sonnet 4.6 agentic)

### 주요 변경

#### 1. 추론 트리 시각화 (Δ4)
AI가 "어떻게 생각했는지" 가시화합니다. 조건 → 보조함수 → derived fact → 답까지의 multi-hop 분해를 풀스크린 트리로 펼쳐볼 수 있습니다. 학생이 막혔던 step 은 ⚠ 노란색, AI가 어려웠던 step 은 빨간색으로 강조됩니다.

#### 2. AI도 어려웠던 순간 (Δ3)
"AI도 이 step 에서 turn 7번, tool 3번 retry, 12.4초 소요" — 정직한 자가 평가로 학생이 동질감을 느낄 수 있습니다. Haiku 4.5 가 1~2문장으로 "어떻게 해결했는가" resolution 을 요약합니다.

#### 3. 5종 quota 통합 (Δ1)
- 하루 5문제 풀이 + 레전드 호출 3회 + R1 리포트 1회
- 주간 리포트 (월요일 reset, 누적 10문제 후)
- 월간 리포트 (1일 reset, 누적 20문제 후)
- 자격 게이트 미충족 시 "n/10 풀이 후 활성화" 안내

#### 4. trigger 확장 카드
같은 발동 조건을 가진 도구 3개를 추천합니다. 예: "도함수 부호 변화 → 극값" 도구를 사용했다면, 같은 trigger 의 다른 도구 (e.g. "이계도함수 → 변곡점") 도 카드로 노출.

### 도메인 변경
기존 `/euler/*` → 자동으로 `/legend/*` 로 리다이렉트됩니다 (301 영구). 즐겨찾기는 그대로 작동합니다.

### Gemini 안정성 (Δ2)
가우스(Gemini 3.1 Pro) 가 일시적 429 한도 초과 시 폰 노이만(GPT-5.5) 으로 자동 fallback 됩니다. UI 에 "가우스가 잠시 휴식 중" 메시지가 표시됩니다.

### 피드백 요청
1주 사용 후 4가지 질문에 답해주시면 감사하겠습니다:

1. **추론 트리 시각화** 가 도움됐나요? (5점 척도)
2. **AI도 어려웠던 순간** 섹션이 동질감을 줬나요? (5점 척도)
3. **trigger 확장 카드** (같은 발동 조건 도구) 가 유용했나요? (5점 척도)
4. **전체 만족도** 는? (5점 척도 + 자유 코멘트)
