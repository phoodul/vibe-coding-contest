# Business Vision — Legend Tutor 사업 비전

## 1. Mission Statement

**우리는 "AI 가 일타강사보다 정답까지 빠르고 정확한 길을 안내하는 시대"를 만든다.**

한국 중·고등학생에게 16개의 정밀하게 가다듬은 AI 도구를 매일 들고 다니는 "AI 학습 동반자" 로 제공한다. 단순히 답을 찾아주는 도구가 아니라, 거장 5명(라마누잔·가우스·폰 노이만·오일러·라이프니츠)의 사고 패턴이 학생의 문제 풀이 과정에 그대로 스며드는 **소크라테스식 심층 코칭** 을 핵심에 둔다. 2~3년 후 학습 시장이 AI 로 재편될 때, "왜 이 길로 가야 하는지"를 명제(Trigger)로 축적해 둔 우리가 일타강사 직관을 가장 잘 언어화한 플레이어가 되어 시장을 선점한다.

> **v1.1 정정 (2026-05-02) — 콴다 vs 우리, 한 줄 차별화**
>
> **콴다는 "문제의 답"을 주고, 우리는 "답에 이르는 길"을 가르친다.**
>
> 콴다의 본질은 "모르는 한 문제를 사진 찍어 올리면 명문대 학생이 풀어 주거나 답지에 매칭" 해 주는 것이다. 즉 단일 문제의 정답·풀이를 검색·전달한다. 반면 우리의 본질은 그 정답에 도달하기 위해 일타강사가 학생의 머릿속에 심어 주는 사고 과정 — **"왜 이 도구를 꺼내야 하는가" (why)** — 를 LLM 으로 명제화하여 매 풀이마다 학생에게 전수하는 것이다. 같은 한 문제를 두고 콴다는 "답을 보여" 주지만 우리는 "다음에 비슷한 문제를 만났을 때 학생 스스로 그 답에 도달할 수 있는 사고의 길" 을 코칭한다. 이 차이가 우리 가격대 (₩50,000~₩300,000) 와 콴다 (₩18,500) 의 카테고리 자체를 가른다.

미션의 측정 가능한 형태는 다음 세 가지다.

1. **Trigger 명제화 자산 50만 개 축적** — 수학에서 검증된 "Tool=A이면 B / Trigger=왜 이 도구"라는 명제 구조를 영어·국어·사회·과학으로 확장한다.
2. **유료 구독 학생 10만 명** — 단일 GTM 채널(수만휘 314만 회원) 자발적 후기로 진입한 뒤 출판사 라이선스로 외부 콘텐츠를 결합한다.
3. **출판사 라이선스 3사 이상** — 천재·동아·비상 중 1순위는 수학 강점인 동아출판, 2순위는 천재교육이다. AI 디지털교과서 붕괴(2025년 8월) 이후 B2C·학원 채널 전환 중이라 협상 타이밍이 최적이다.

## 2. 시장 가설

### 2-1. 핵심 공백 — "수능 심층 코칭" 카테고리는 비어 있다

이 공백 — **"한국 수능 본문에 특화된 AI 1:1 심층 코칭"** — 이 우리의 진입 위치다. Legend Tutor 의 KPI 89.5% (Gemini 3.1 Pro agentic 38문항 multi-turn) 가 이 공백을 입증한다.

### 2-2. 결정적 타이밍 — 출판사 B2B2C 전환

2024~2025년 한국 정부와 12개 출판사가 ₩1.2조+₩8,000억을 투입해 추진한 **AI 디지털교과서 사업이 2025년 8월 국회 법안 통과로 법적 지위가 박탈** 됐다. 채택률은 37% → 19% 로 급락했고, 출판사들은 학원 파트너십과 B2C 구독 앱으로 선회 중이다 (research_raw.md §3-1).

이 흐름이 우리에게 의미하는 바는 명확하다.

- 천재교육·동아출판은 이미 디지털 코어 자산을 만들어 두었고 **B2B2C 채널을 적극 모색** 중이다.
- 콘텐츠 라이선스 분배율 협상에서 출판사 측이 **양보 가능한 카드** 를 가지고 있다 (B2G 채널 봉쇄로 대안 제한).
- AI 인강 + AI 코칭을 **하나의 앱에서 묶어 주는 플레이어가 없다.** 우리가 그 위치를 선점할 수 있다.

### 2-3. 2~3년 시점 시장 재편 시나리오

각 시나리오는 2027~2028년 시점, 한국 K-12 AI 학습 시장에서 우리 위치를 가정한다. 모두 **추정** 이다.

#### Bull 시나리오 — 카테고리 1위 진입 + 인강 거인 역전

> **v1.1 정정 (2026-05-02) — 메가스터디 역전 시나리오 명시**
>
> 수년 안에 메가스터디·시대인재·이투스 같은 한국 인강 거인이 AI 기반 신생사에 시장 점유율을 빼앗기는 시나리오는 우리에게 강한 가설이다. 콴다·뤼튼처럼 단순 검색·범용 LLM 만 가진 회사는 인강 거인을 대체하지 못한다. 인강 거인을 대체할 수 있는 것은 **"정답 도달 핵심(why) 코칭" 자산을 보유한 회사** 다. 일타강사가 학생에게 5초 안에 떠올려 주는 "이 도구를 꺼내라" 의 명제(Trigger) 가 50만 개 누적되면, 학생 입장에서 메가패스 영상 50시간보다 우리의 1:1 코칭 1시간이 더 효율적이다. 우리는 그 역전의 후보 중 하나로 시장 선점이 목표다.

- 수만휘 자발적 후기 1만+ 누적 → 콴다 대비 "심층 코칭 = Legend Tutor" 인식 정착.
- 출판사 1~2개 사 정식 라이선스 (천재 또는 동아) → AI 인강 + AI 코칭 통합 앱 발매.
- **메가스터디·시대인재 등 인강 거인이 AI 신생사에 시장 점유율을 뺏김.** 콴다·뤼튼 같은 검색·범용 AI 가 아닌, **정답 도달 핵심(why) 코칭** 자산을 가진 회사가 그 자리를 차지한다 — 우리는 그 후보군의 1 ~ 3위 중 하나.
- **유료 사용자 10만+ / ARR 200~300억 / 출판사 분배 후 마진 60%+**
- 학원 채널: 직영 학원 30개 + 가맹 100개 PoC 가동.
- **시장 점유율 추정**: 한국 K-12 AI 학습 시장 (2028년 추정 5,000억) 중 5~6%.

#### Base 시나리오 — 자체 B2C 안착

- 수만휘 후기 누적 3,000건 + 인플루언서 협업 5건 → 콴다 대비 인지도 30% 수준.
- 출판사 PoC 진행 중 (정식 계약 1건 미만).
- **유료 사용자 5만 / ARR 30~50억 / 마진 70%+**
- 학원 채널: PoC 5~10개.
- **시장 점유율 추정**: 1~2%.

#### Bear 시나리오 — 차별화 약화

- 콴다·EBS 가 심층 코칭 기능을 추가하며 commoditization → 우리 차별화 약화.
- 출판사 협상 결렬 또는 다른 플레이어와 독점 계약.
- **유료 사용자 5,000~1만 / ARR 5~10억** — 베타 + 초기 유료 외 성장 정체.
- 대안: 1과목(수학) 특화 prosumer SaaS 로 포지셔닝 변경.

## 3. 차별화 무기

### 3-1. Trigger 라이브러리 — 일타강사 직관의 명제화

핵심 차별점은 **Trigger 라이브러리** 다. 13차 세션에서 사용자가 직접 정립한 정의:

- **Tool** = "A 이면 B" 라는 명제 (예: "log 가 보이면 → 밑변환 공식 후보")
- **Trigger** = "왜 이 도구인가?" 의 답 (예: "지수 ↔ 로그 양방향 변환에서 막힐 때")

수학에서 6 anchor seed → 244 도구 / 463 trigger / 926 임베딩이 검증됐다. 이는 **"일타강사가 학생을 보자마자 5초 안에 떠올리는 풀이의 길" 을 검색 가능한 자산으로 변환** 한 것이다. 이 자산이 50만 개로 누적되는 순간, 어떤 경쟁사도 단기간에 따라잡을 수 없는 데이터 해자(moat)가 된다.

### 3-2. 5거장 페르소나 + 소크라테스식 코칭

- **라마누잔** — 직관·패턴 발견
- **가우스** — 우아한 일반화
- **폰 노이만** — 계산·추정·근사
- **오일러** — 계산 기교·시리즈
- **라이프니츠** — 표기·기호 설계

3-Stage Adaptive Router 가 학생 풀이의 막힘 지점을 분석해 적절한 거장을 호출한다. R1 풀이 정리 카드 (schema 1.4) 가 한 문제 풀이 종료 시 자동 발급된다. 이 경험은 콴다·EBS·메가스터디 어느 곳에서도 재현 불가능하다.

### 4-1. 양 트랙 흐름도

### 4-2. B2C 직접

- **타깃**: 고2~3 N수생 + 학원에 추가로 AI 코칭이 필요한 중·고등학생.
- **결제**: Toss Payments 메인 + 카카오페이 + 휴대폰 본인인증.
- **수익 흐름**: 월 구독 (Lite ₩50,000 / Standard ₩150,000 / Pro ₩300,000 / Family ₩400,000).
- **마진 추정**: research_raw.md §4-5 의 ₩11,088 mixed 라우팅 + Caching 40% 절감 → 토큰 비용은 구독료의 15~25%, 마진 70~80% 유지 (자세한 계산은 [pricing-strategy.md](./pricing-strategy.md) §3).

### 4-3. B2B2C 출판사 라이선스

- **타깃**: 천재·동아·비상·YBM 등 K-12 출판사.
- **콘텐츠**: 출판사 교과서·문제집·기출 → OCR + Trigger 라이브러리 자동 매핑 → AI 인강 모드 + AI 코칭 모드 통합.
- **분배 구조 추정** (research_raw.md §3-4 음원 7:3 / 전자책 25회 80% 모델 참고):
  - 플랫폼 (우리) **65~75%**
  - 출판사 **25~35%**
  - 콘텐츠 단위 (책별 / 챕터별) 협상 가능
- **DRM**: chapter-level signed URL + watermark (자세한 사양은 [architecture-platform.md](./architecture-platform.md) §2).
- **데이터 피드백**: 학생 풀이 데이터 → 출판사 통계 대시보드 → 교재 개선 인사이트. **이 피드백 루프가 출판사 협상의 핵심 카드** 다.

## 5. 시장 진입 (GTM) 전략

### Phase 1 — 수만휘 자발적 후기 + 텐볼스토리 파트너십 (즉시 ~ 4주)

**핵심 가설**: "정회원 자격 베타 사용자 5명 → 자발적 후기 게시 → 30~80명/건 가입 유입" (H1).

1. 수만휘 정회원 자격 보유 수험생 5명을 슬랙·인스타로 모집 → 30일 무료 베타.
2. 일주일에 1회 "사용 일지" 가이드 제공 (강요 X, 자율 게시 권장).
3. 운영사 텐볼스토리에 별도 제안서 발송 — "수만휘 클래스 인강 + Legend Tutor 코칭 통합 PoC".
4. 동시에 오르비 정회원 가입 후 "N수생 수능 수학 도전기" 게시.

**예산**: 베타 사용자 5명 × 30일 무료 = 토큰 비용 ₩50,000~₩100,000. 마케팅 비용 0.

**KPI**: 수만휘 자발 후기 3건 + 오르비 후기 2건 + 텐볼스토리 회신 1건.

### Phase 2 — 인플루언서 + 학원 PoC (3~6개월)

- 입시 유튜버 5명 협찬 (편당 100~500만 추정, research_raw.md §6-4) — Legend Tutor 단독 사용 영상.
- 학원 5~10곳 PoC: 학원 강사 1명에게 무료 Pro 계정 + 학생 30명 무료 Lite → 학원 추천 효과 측정.
- 인스타·틱톡 SNS 마케팅 (Z세대 학생 직접 접점).

**예산**: 인플루언서 협찬 1,500만 + 학원 PoC 토큰 비용 500만 ≈ 2,000만/분기.

**KPI**: 유료 사용자 1,000명 / 학원 정식 계약 1건.

### Phase 3 — 출판사 정식 라이선스 (6~12개월)

- 천재교육·동아출판·비상교육에 미팅 제안.
- PoC 1권 (영어 문법 또는 수학 1단원) → 3개월 사용 데이터 → 정식 계약.
- 분배율 협상 (플랫폼 70% / 출판사 30% 목표).

**예산**: 법무 자문 + 협상 인력 = 5,000만 (1차 개발자·법무 채용 시점).

**KPI**: 출판사 PoC 합의 1건 + 정식 계약 1건.

### Phase 4 — 본격 출시 + 1차 채용 (12~24개월)

- 정식 출시 마케팅 (1억+).
- 풀타임 개발자 3~5명 + UX 디자이너 1명 + 콘텐츠 운영 2명.
- 학원 가맹 50개 + 직영 5개.

자세한 분기별 일정은 [roadmap.md](./roadmap.md) Phase 5 참조.
