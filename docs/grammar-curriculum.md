## 컨셉

**한국 영문법 표준 13 대단원 × 4~6 레슨 = 75 레슨 텍스트북.**
영어 단어 학습 (18,000 단어 에베레스트 등반) 과 동일한 커리큘럼 학습 모델.

각 레슨 = 한국어 설명 800~1500자 + 영어 대표 문장 1개 (외우기) + 4지선다/빈칸 실전 5문제 + 해설.

학년 매핑: 중1~고3 (수능·내신 직결).
