# Pricing Strategy — Tier 설계와 토큰 경제

## 1. 가격 철학

> **v1.1 정정 (2026-05-02) — 가격 정당화의 핵심 framing**
>
> **우리 Pro ₩300,000 / Standard ₩150,000 은 "새로운 비싼 SaaS" 가 아니다. 한국 학부모가 이미 매월 지불하고 있는 "단과 학원 1과목 자리" 에 들어가는 AI 코치다.**
>
> 한국 사교육비의 표준은 **단과 학원 1과목 월 ₩20~30만** 이다. 학부모는 자녀가 수학 단과 ₩25만 + 영어 단과 ₩20만 + 과학 단과 ₩20만 = 월 ₩65만 식으로 이미 지불하고 있다. 우리 Pro ₩30만 은 그 단과 한 자리를 대체하는 가격이다. 하루 2~3시간 AI 수업 = 단과 학원 시간과 동일하면서, 24/7 접근성 + 무제한 횟수 + 5거장 페르소나 코칭이 추가된다.
>
> 즉 우리는 "₩30만 SaaS 를 새로 사세요" 가 아니라 **"이미 쓰고 있는 단과 학원 1과목 자리를 AI 코치로 바꾸세요"** 라고 부모에게 말한다. 이 framing 이 가격 저항을 본질적으로 우회한다.

### 1-1. 카테고리 정의

우리는 **일반 SaaS 가 아니다**. AI 인강 + 1:1 코칭 카테고리에서 "거장 5명 페르소나가 학생과 대화하는 심층 학습 경험" 을 제공한다.

| 비교군 | 가격대 (월) | 구조 | 우리 Tier 매핑 |
|---|---|---|---|
| 콴다 프리미엄 | ₩18,500 | 문제풀이 검색 | (카테고리 다름 — 답 검색 vs why 코칭) |
| 산타토익 | ₩22,000 ~ ₩43,000 | 토익 점수 예측 | — |
| 메가패스 인강 (연 환산) | 약 ₩56,000 | 강사 영상 무제한 (편당 가치 분산) | Lite ₩50,000 동급 가격대 |
| **단과 학원 1과목 (수학·영어·과학 1개)** | **₩20~30만** | 그룹 강의 1과목 / 주 2~3회 | **Standard ₩150,000 ~ Pro ₩300,000 = 단과 1자리** |
| **단과 학원 2과목 (수학+영어 등)** | **₩40~60만** | 그룹 강의 2과목 | **Family ₩400,000 = 단과 2자리** |
| 1:1 과외 (시간당 ₩5~15만 × 월 8회) | **₩40~120만** | 강사 1:1 | Pro 1/4 가격 |
| 대치동 종합 학원 | **₩50~150만** | 오프라인 그룹 강의 | Pro 1/3 ~ 1/5 가격 |
| 학원 + 과외 패키지 | **₩200만+** | 풀스택 | (대체 후보 X — 너무 비싼 시장) |

(출처: research_raw.md §5-4 + 단과 학원 ₩20~30만 baseline 은 한국 사교육비 통상 가격)

**핵심 framing**: 우리 가격은 "새로운 SaaS 비용" 이 아니라 **"이미 부모가 매월 쓰고 있는 단과 학원 1과목 자리"** 다. Pro ₩30만 은 단과 한 과목 평균 가격대에 정확히 위치한다. 즉 부모는 새 지출을 만들 필요 없이, 현재 쓰는 단과 1자리를 AI 코치로 대체하면 된다. 그 결과는 동일 시간대 + 무제한 횟수 + 24/7 접근성 + 5거장 페르소나.

### 1-2. 토큰 비용 흡수 원칙

고품질 AI 호출 (Sonnet/Opus + agentic multi-turn + Gemini 3.1 Pro) 이 핵심 가치다. 콴다 ₩18,500 가격대에서는 Haiku 만으로 답변할 수밖에 없어 "왜 정답인지" 심층 코칭이 구조적으로 불가능하다. **5~30만원 가격대만이 Sonnet/Opus 충분 사용을 허용** 한다 (research_raw.md §4-5 토큰 경제 표).

### 1-3. 학생 자발 결제 vs 부모 결제 분리

research_raw.md §5-4 에 따르면 **학생 자발 결제 상한선은 월 2~5만원** 이다. 5만원 초과는 부모 설득이 필수다. 따라서 우리 tier 는 다음과 같이 분리한다.

- **Free / Lite** = 학생 자발 결제 OK (고2~3 본인 카드)
- **Standard / Pro / Family** = 부모 설득 마케팅 필수

## 2. Tier 설계

### 2-1. Tier 요약 표

| Tier | 가격 (월) | 모델 라우팅 | 일 사용 한도 | 거장 페르소나 | 주요 기능 |
|---|---|---|---|---|---|
| **Free** | ₩0 (14일) | Haiku 100% | 1일 10턴 | 1명 (선택) | 가드레일 풀, 베타 검증 |
| **Lite** | ₩50,000 | Haiku 80% / Sonnet 20% | 1일 100턴 | 1명 선택 | 16 도구 전부 + Trigger 라이브러리 검색 |
| **Standard** | ₩150,000 | Haiku 50% / Sonnet 40% / Opus 10% | 무제한 | 5명 전부 | + 수능 정답 해석 + 메타인지 카드 + 부모 리포트 |
| **Pro** | ₩300,000 | Haiku 30% / Sonnet 50% / Opus 20% + Gemini 3.1 Pro agentic | 무제한 | 5명 + 신규 거장 | + 출판사 콘텐츠 우선 액세스 + 학원 매칭 + 1:1 멘토링 |
| **Family** | ₩400,000 (학생 2명) | Pro 동일 | 무제한 | 5명 전부 | + 부모 대시보드 + 자녀 비교 |
| **Academy** | 별도 견적 | Pro 동일 | 무제한 | 5명 전부 | + 학원 운영 도구 + 학생 30~300명 단위 |

### 2-2. Tier 별 학습 효과 추정

각 Tier 의 모델 라우팅이 만드는 학습 경험 차이.

| Tier | 학습 경험 | 학습 효과 추정 |
|---|---|---|
| Free | 기초 개념 질의응답 (Haiku 한정) | 콴다 무료 수준 |
| Lite | 1과목 1거장 코칭 + 16 도구 활용 | 콴다 프리미엄 + 일부 심층 코칭 |
| Standard | 5거장 자유 호출 + 수능 정답 해석 + 메타인지 코칭 | 학원 그룹 강의 70% 대체 가능 |
| Pro | 모든 기능 + agentic multi-turn (Gemini 3.1 Pro) + 출판사 콘텐츠 통합 | 1:1 과외 50~70% 대체 가능 |
| Family / Academy | Pro + 부모/학원 대시보드 | 학원 부분 대체 + 학원 효율화 |

(추정 — 학습 효과는 베타 데이터 누적 후 정량화 예정)

### 2-3. Free → Lite → Standard → Pro 업셀 트리거

- Free → Lite: 14일 후 자동 만료. "지금 Lite 시작하면 Trigger 라이브러리 무제한 검색"
- Lite → Standard: 1일 사용량 100턴 도달 빈도 ≥ 3회/주 → "무제한으로 전환하세요" 알림
- Standard → Pro: 수능 D-100 카운트다운 시점 → "수능 D-Day Pro 모드: agentic 멀티턴" 마케팅
- Pro → Family: 자녀 2명 가구 시점 → "Family 가 Pro 2개보다 33% 저렴"

## 3. 토큰 경제 분석

### 3-1. 베이스 가정 (research_raw.md §4-5 인용)

- 학생 1인 월 사용량: 일 1시간 × 22일 = 660 메시지
- 메시지당 input 2,000 토큰 / output 800 토큰
- 월 input 1.32 MTok / output 0.528 MTok
- 1$ = ₩1,400

### 3-2. Tier 별 모델 라우팅 비용

#### Lite (Haiku 80% + Sonnet 20%)

```
input  = 0.8 × $1.32 (Haiku) + 0.2 × $3.96 (Sonnet) = $1.06 + $0.79 = $1.85
output = 0.8 × $2.64 (Haiku) + 0.2 × $7.92 (Sonnet) = $2.11 + $1.58 = $3.69
월 합계 = $5.54 ≈ ₩7,756
```

Caching 40% 적용 (시스템 프롬프트 + 거장 페르소나 5분 TTL):
- input 부분 50% 가량이 캐시 hit → input 비용 ₩2,590 → ₩1,295 (50% 절감)
- 월 합계 ≈ **₩6,460**

#### Standard (Haiku 50% + Sonnet 40% + Opus 10%)

```
input  = 0.5 × $1.32 + 0.4 × $3.96 + 0.1 × $6.60 = $0.66 + $1.58 + $0.66 = $2.90
output = 0.5 × $2.64 + 0.4 × $7.92 + 0.1 × $13.20 = $1.32 + $3.17 + $1.32 = $5.81
월 합계 = $8.71 ≈ ₩12,194
```

학생 무제한 (실 사용량 1.5x 가정 = 일 1.5시간):
- ₩12,194 × 1.5 = ₩18,291
- Caching 40% → **₩14,633**

#### Pro (Haiku 30% + Sonnet 50% + Opus 20% + Gemini 3.1 Pro agentic 10턴/일)

```
Anthropic 부분:
input  = 0.3 × $1.32 + 0.5 × $3.96 + 0.2 × $6.60 = $0.40 + $1.98 + $1.32 = $3.70
output = 0.3 × $2.64 + 0.5 × $7.92 + 0.2 × $13.20 = $0.79 + $3.96 + $2.64 = $7.39
Anthropic 월 합계 = $11.09

Gemini 3.1 Pro agentic 10턴/일:
input  = 22일 × 10턴 × 4,000토큰 (multi-turn) = 0.88 MTok × $2.00 = $1.76
output = 22일 × 10턴 × 1,500토큰 = 0.33 MTok × $12.00 = $3.96
Gemini 월 합계 = $5.72

전체 합계 = $11.09 + $5.72 = $16.81 ≈ ₩23,534
```

학생 무제한 + agentic (실 사용량 2x 가정):
- ₩23,534 × 2 = ₩47,068
- Caching 40% → **₩37,654**

### 3-3. Tier 별 마진 계산표

수익 구조: 구독료 - 토큰 비용 - 결제 수수료(3%) - 인프라 (Vercel + Supabase + 운영)

| Tier | 구독료 | 토큰 비용 | 결제 수수료 (3%) | 인프라 | 마진 (월) | 마진율 |
|---|---|---|---|---|---|---|
| Lite ₩50,000 | 50,000 | 6,460 | 1,500 | 500 | **41,540** | **83%** |
| Standard ₩150,000 | 150,000 | 14,633 | 4,500 | 1,500 | **129,367** | **86%** |
| Pro ₩300,000 | 300,000 | 37,654 | 9,000 | 3,000 | **250,346** | **83%** |
| Family ₩400,000 (학생 2명) | 400,000 | 75,308 (Pro × 2) | 12,000 | 3,000 | **309,692** | **77%** |

**핵심**: 모든 Tier 가 70% 이상 마진을 유지한다. 가격 5만원 (Lite) 에서도 Caching 효과로 토큰 비용은 13% 수준이다.

### 3-4. 사용량 폭증 리스크 — Cost Guard 설계

학생이 1일 100~200턴을 매일 사용하는 극단 사용자 (heavy user) 가 발생하면 Pro 라도 토큰 비용이 ₩100,000 이상으로 급등할 수 있다. 대응:

| 단계 | 트리거 | 액션 |
|---|---|---|
| 1단계 | 일 80턴 도달 | "오늘 충분히 학습하셨습니다" 알림 |
| 2단계 | 일 150턴 도달 | Sonnet/Opus → Haiku 자동 다운그레이드 |
| 3단계 | 월 토큰 비용 ₩50,000 (Lite) / ₩150,000 (Pro) 초과 | Sonnet/Opus 호출 차단 (Haiku 만 허용) + 운영팀 알림 |
| 4단계 | 월 토큰 비용 구독료의 50% 초과 | 사용자에게 1:1 알림 + 사용 패턴 협의 (학원 모드 제안) |

자세한 구현은 [architecture-platform.md](./architecture-platform.md) §5 참조.

## 4. 학생 자발 vs 부모 결제 마케팅

> **v1.1 정정 (2026-05-02) — 부모에게 던지는 한 문장**
>
> 부모 마케팅의 핵심 한 문장은 이것이다.
>
> **"새 지출을 만들지 마세요. 자녀가 다니는 단과 학원 한 자리를 AI 코치로 바꾸세요."**
>
> 한국 학부모는 이미 자녀에게 단과 학원 1~3과목 (월 ₩20~30만 × 2~3 = ₩40~90만) 을 지불하고 있다. 우리 Standard ₩150,000 / Pro ₩300,000 는 그 중 1과목을 대체하는 가격이다. 즉 우리는 "AI 코칭" 이라는 새 카테고리에 부모를 끌어오는 것이 아니라, **이미 부모가 쓰고 있는 사교육비 안에서 한 자리를 우리가 가져오는 것** 이다. 이 framing 이 부모 결제 저항의 80%를 우회한다 (가설, Phase 3 결제 출시 후 검증).

### 4-1. 결제 행태 가설 (research_raw.md §5-4 인용)

- **학생 자발 결제 가능선**: 월 2~5만원 (콴다 ₩18,500 가까운 영역)
- **부모 결제 비중 추정**: 5만원 초과 구독은 부모 결제 70%+ (정확 통계 미발견, 추정)
- **고2~3 본인 결제**: 만 18세 이상 본인 카드 발급 가능 → Lite ₩50,000 자발 결제 OK
- **만 14세 미만**: 부모 카드 결제 + 부모 동의 필수 (법적 의무)
- **부모가 이미 쓰고 있는 사교육비 baseline**: 단과 학원 1과목 ₩20~30만 × 2~3 과목 = 월 ₩40~90만. **우리는 그 중 1자리에 들어간다.**

### 4-2. Lite (₩50,000) — 학생 자발 결제 마케팅

**메시지**: "학원 1회 안 가는 비용으로 24/7 1:1 코칭"

- 수만휘 자발적 후기 → 입소문 (research_raw.md §1-5 콴다 사례)
- 인스타·틱톡 짧은 콘텐츠 (학생이 본인 사용 영상 게시)
- "수능 D-100 챌린지" 무료 14일 → 자동 Lite 전환

### 4-3. Standard / Pro (₩150,000 ~ ₩300,000) — 부모 설득 마케팅

부모 결제는 **3가지 자료** 가 핵심이다.

#### 자료 1: 학원 대비 비용 절감 ROI 계산기 — "단과 1자리 대체" framing

부모 입장에서 가장 강한 메시지는 "새 돈 쓰지 마라" 다. 두 가지 시나리오로 ROI 를 보여 준다.

**시나리오 A — 단과 1과목 대체 (가장 자연스러운 진입)**
```
현재: 수학 단과 학원 ₩25만 + 영어 단과 ₩20만 + 과학 단과 ₩20만 = 월 ₩65만
           ↓
변경: 수학 단과 ₩25만 → Pro ₩30만 (AI 코치) 로 대체
           ↓
월 비용: ₩65만 → ₩70만 (+₩5만)
       하지만 24/7 무제한 + 5거장 페르소나 + 메타인지 코칭 추가
       (사실상 단과 1과목 + 1:1 과외 + 24시간 자율학습실 가치)
```

**시나리오 B — 학원 + 과외 풀스택 (대체 비용 큼)**
```
학원 월 ₩100만 + 1:1 과외 월 ₩40만 = ₩140만
                         ↓
Pro ₩30만 (학원 대체 60% + 24/7 추가)
                         ↓
연간 절감: (140 - 30) × 12 = ₩1,320만
```

부모 페이지에 직접 입력 가능한 ROI 계산기. **시나리오 A 를 디폴트로 보여 주는 것** 이 핵심 — "새 비용" 이 아니라 "단과 1자리 대체" framing.

#### 자료 2: 자녀 사용 리포트 (월간 PDF)

- 학습 시간 / 풀이 수 / 강점·약점 영역 (시각화 그래프)
- "가장 많이 호출한 거장: 가우스" 등 학습 스타일 인사이트
- 위기 신호 횟수 (가드레일 트리거 통계)
- 동급 학년 평균 대비 위치 (베타 데이터 누적 후)

#### 자료 3: 학습 효과 통계 (Phase 1 베타 종료 후)

- "Pro 사용 3개월 후 모의고사 평균 X점 상승" (베타 50명 데이터)
- KPI 89.5% 입증 자료 (이미 보유)
- Khan Academy 대비 추가 학습 효과 20~30% (research_raw.md §2-9)

### 4-4. Family Tier 마케팅

자녀 2명 이상 가구 ROI: ₩40만 / 2명 = 1인당 ₩20만 (Standard 보다 저렴 + Pro 기능).

학원 대체 효과: 자녀 2명이 학원 1개씩 다닐 경우 월 ₩200만 → ₩40만으로 80% 절감.

## 5. 결제 인프라

### 5-1. Toss Payments 메인

- 결제 수수료 2.9% + 부가세 (한국 본사, 안정성)
- 정기결제 자동 갱신 지원
- 환불 정책: 7일 내 100% 환불 (소비자기본법)

### 5-2. 카카오페이 보조

- 결제 수수료 2.9% (Toss 동일)
- 카카오 로그인 연동으로 회원가입 1step 단축
- 14~18세 '틴즈넘버' 서비스 (research_raw.md §5-1) — 1회 ₩20만 한도, 월 ₩100만 한도. **Standard 결제 가능 / Pro 가능 / Family 한도 초과로 부모 카드 필요**

### 5-3. 휴대폰 본인인증

- PASS 앱 (한국 통신 3사 통합)
- 만 14세 이상 자기 명의 휴대폰 보유 시 사용 가능
- 부모 동의 의무 (만 14세 미만) 시 부모 휴대폰 본인인증 추가

### 5-4. 만 14세 미만 부모 동의 플로우

```
[학생 가입 시도]
       │
       ▼
[생년월일 입력] ──── 만 14세 이상 ───→ [본인인증 → 결제]
       │
       │ 만 14세 미만
       ▼
[부모 이름·연락처 입력]
       │
       ▼
[부모 SMS 인증 (본인 명의 폰)]
       │
       ▼
[부모 이메일 동의 링크 클릭]
       │
       ▼
[부모 카드 등록 (부모 명의 카드)]
       │
       ▼
[학생 가입 완료 + 부모 대시보드 자동 활성화]
```

자세한 DB 스키마는 [architecture-platform.md](./architecture-platform.md) §7 참조.

## 6. 가격 실험 계획

### 6-1. 단계별 출시 가격

| 단계 | 시기 | 가격 정책 |
|---|---|---|
| 베타 (Phase 0~2) | 즉시 ~ 8주 | 모든 사용자 무료 (30일 만료 정책 유지) |
| 초기 유료 (Phase 3) | 9주차 ~ 12주차 | Lite ₩50,000 + Standard ₩150,000 만 출시 (Pro 보류) |
| 정식 출시 (Phase 5) | 24주차 ~ | 4 tier 전체 + Family + Academy |

**Pro Tier 6개월 보류 이유**: agentic multi-turn 토큰 비용이 가장 변동성 크다. 베타 + Standard 사용 데이터로 토큰 사용량 패턴 파악 후 Pro 가격 확정.

### 6-2. A/B 가격 테스트

Phase 3 출시 후 다음 변형 A/B 테스트.

| 실험 | 변형 A | 변형 B | 측정 |
|---|---|---|---|
| Lite 가격 | ₩50,000 | ₩30,000 | 가입율 / 마진 |
| Standard 무료 체험 | 7일 | 14일 | 전환율 |
| 부모 ROI 계산기 위치 | 가입 페이지 | 1주일 사용 후 이메일 | Standard → Pro 업셀율 |
| 거장 5명 vs 1명 차등 | Lite=1명 / Standard=5명 | Lite=5명 / Standard=5명 (구분 X) | NPS / 업셀율 |

### 6-3. 가격 인하 시나리오 (Bear 시나리오 대응)

Phase 3 결제 출시 3개월 후 유료 전환율 < 5% 시 다음 순서로 검토.

1. Lite ₩50,000 → ₩30,000 (학생 자발 결제선 진입)
2. Standard ₩150,000 → ₩90,000
3. Pro ₩300,000 유지 (프리미엄 포지션)
4. Free 14일 → 30일 연장 (체험 완성도 강화)

### 6-4. 가격 인상 시나리오 (Bull 시나리오)

출판사 정식 라이선스 체결 후:

1. Pro ₩300,000 → ₩400,000 (출판사 콘텐츠 추가 가치)
2. 신규 Tier "Legend ₩500,000" 출시 (1:1 인간 멘토 매칭)
3. Lite ₩50,000 유지 (대중 진입선)
