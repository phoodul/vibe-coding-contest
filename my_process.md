# vibe-coding-contest

## 사전 준비 과정
1. 공모전 관련 내용을 Gemini에게 물어보고 우승 전략을 정리하게 했다. 
2. GitHub에서 remote first 방식으로 사전에 repository를 만들었다. 
3. .gitignore 파일을 node로 설정하고, 추가로 전략에 맞게 수정했다. 
4. 목적에 맞는 rule, skill, workflow 설정한다. (Gemini Pro에게 검색)
5. rule에는 Andrej Karpathy의 코딩 철학을 추가했다.
6. Claude MCP에 supabase, playwright, sequential thinking, Context7, Firebase, Vercel을 설치했다.
7. supabase token을 .env에 저장했다.
8. Firebase mcp를 설치하고, serviceAccountKey.json을 .env에 저장했다.
9. Vercel OAuth를 설정했다.
